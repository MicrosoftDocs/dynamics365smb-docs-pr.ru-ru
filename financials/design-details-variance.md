# <a name="design-details-variance"></a><span data-ttu-id="4bc23-103">Сведения о проектировании: отклонение</span><span class="sxs-lookup"><span data-stu-id="4bc23-103">Design Details: Variance</span></span>
<span data-ttu-id="4bc23-104">Отклонение определяется как разница между фактическими и стандартными затратами, как описано в следующей формуле.</span><span class="sxs-lookup"><span data-stu-id="4bc23-104">Variance is defined as the difference between the actual cost and the standard cost, as described in the following formula.</span></span>  

 <span data-ttu-id="4bc23-105">фактическая себестоимость - стандартная себестоимость = отклонение</span><span class="sxs-lookup"><span data-stu-id="4bc23-105">actual cost – standard cost = variance</span></span>  

 <span data-ttu-id="4bc23-106">Если фактическая себестоимость меняется, например, в результате учета товарных издержек в более позднюю дату, отклонение обновляется соответствующим образом.</span><span class="sxs-lookup"><span data-stu-id="4bc23-106">If the actual cost changes, for example, because you post an item charge on a later date, then the variance is updated accordingly.</span></span>  

> [!NOTE]  
>  <span data-ttu-id="4bc23-107">Переоценка не влияет на расчет отклонения, поскольку переоценка меняет только стоимость запасов.</span><span class="sxs-lookup"><span data-stu-id="4bc23-107">Revaluation does not affect the variance calculation, because revaluation only changes the inventory value.</span></span>

 <span data-ttu-id="4bc23-109">В следующем примере показано вычисление вариации для приобретенных товаров.</span><span class="sxs-lookup"><span data-stu-id="4bc23-109">The following example illustrates how variance is calculated for purchased items.</span></span> <span data-ttu-id="4bc23-110">Он основан на следующем сценарии:</span><span class="sxs-lookup"><span data-stu-id="4bc23-110">It is based on the following scenario:</span></span>

5. <span data-ttu-id="4bc23-118">Пользователь переоценивает товар со 100,00 до 70,00 рублей.</span><span class="sxs-lookup"><span data-stu-id="4bc23-118">The user revalues the item from LCY 100.00 to LCY 70.00.</span></span> <span data-ttu-id="4bc23-119">Это не влияет на вычисление отклонений — только на стоимость запасов.</span><span class="sxs-lookup"><span data-stu-id="4bc23-119">This does not affect the variance calculation, only the inventory value.</span></span>
