# <a name="how-to-process-sales-returns-or-cancellations"></a><span data-ttu-id="78664-103">Практическое руководство. Обработка возвратов продажи или отмен</span><span class="sxs-lookup"><span data-stu-id="78664-103">How to: Process Sales Returns or Cancellations</span></span>
<span data-ttu-id="78664-104">Если клиент желает вернуть товары или получить возмещение за товары или услуги, которые вы ему продали и получили оплату, необходимо создать и учесть кредит-ноту продажи, определяющую требуемое изменение.</span><span class="sxs-lookup"><span data-stu-id="78664-104">If a customer wants to return or be reimbursed for items or services that you have sold and received payment for, you must create and post a sales credit memo that specifies the requested change.</span></span> <span data-ttu-id="78664-105">Чтобы включить правильную информацию о счете продажи, вы можете создать кредит-ноту продажи из учтенного счета продажи или использовать функцию копирования.</span><span class="sxs-lookup"><span data-stu-id="78664-105">To include the correct sales invoice information, you can create the sales credit memo from the posted sales invoice, or use a copy function.</span></span>  

> [!NOTE]  
>   <span data-ttu-id="78664-106">Если учтенный счет продажи еще не оплачен, можно использовать функции **Исправить** или **Отмена** в учтенном счете продажи для сторнирования транзакций.</span><span class="sxs-lookup"><span data-stu-id="78664-106">If a posted sales invoice has not yet been paid, then you can use the **Correct** or **Cancel** functions on the posted sales invoice to reverse transactions.</span></span> <span data-ttu-id="78664-107">Эти функции работают только для неоплаченных счетов, они не поддерживают частичные возвраты или отмены.</span><span class="sxs-lookup"><span data-stu-id="78664-107">These functions work only for unpaid invoices, and they do not support partial returns or cancellations.</span></span> <span data-ttu-id="78664-108">Дополнительные сведения см. в разделе [Практическое руководство. <span data-ttu-id="78664-109">В дополнение к исходному учтенному счету продажи можно применить кредит-ноту продажи к другим документам продажи, например к другому учтенному счету продажи, поскольку клиент также возвращает товары, доставленные по этому счету.</span><span class="sxs-lookup"><span data-stu-id="78664-109">In addition to the original posted sales invoice, you can apply the sales credit memo to other sales documents, for example another posted sales invoice, because the customer is also returning items delivered with that invoice.</span></span>

<span data-ttu-id="78664-110">Возврат или возмещение могут быть связаны только с частью товаров или услуг в первоначальном счете продажи.</span><span class="sxs-lookup"><span data-stu-id="78664-110">A return or reimbursement may relate to only some of the items or services on the original sales invoice.</span></span> <span data-ttu-id="78664-111">В этом случае вы должны изменить информацию в строках в кредит-ноте продажи.</span><span class="sxs-lookup"><span data-stu-id="78664-111">In that case, you must edit information on the lines on the sales credit memo.</span></span> <span data-ttu-id="78664-112">Когда вы учитываете кредит-ноту продажи, документы продажи, на которые влияет это изменение, сторнируются и для клиента может быть создано возмещение.</span><span class="sxs-lookup"><span data-stu-id="78664-112">When you post the sales credit memo, the sales documents that are affected by the change are reversed and a refund payment can be created for the customer.</span></span>

<span data-ttu-id="78664-114">Учет кредит-ноты также приведет к отмене всех товарных издержек, которые были назначены учтенному документу, таким образом, что операции стоимости товара будут совпадать со значениями до назначения товарных издержек.</span><span class="sxs-lookup"><span data-stu-id="78664-114">The credit memo posting will also revert any item charges that were assigned to the posted document, so that the item’s value entries are the same as before the item charge was assigned.</span></span>

## <a name="to-create-a-sales-credit-memo-from-a-posted-sales-invoice"></a><span data-ttu-id="78664-115">Создание кредит-ноты продажи из учтенного счета продажи</span><span class="sxs-lookup"><span data-stu-id="78664-115">To create a sales credit memo from a posted sales invoice</span></span>

2. <span data-ttu-id="78664-117">В окне **Учтенные счета продажи** выберите учтенный счет продажи, который необходимо сторнировать, а затем выберите действие **Создать корректирующую кредит-ноту**.</span><span class="sxs-lookup"><span data-stu-id="78664-117">In the **Posted Sales Invoices** window, select the posted sales invoice that you want to reverse, and then choose the **Create Corrective Credit Memo** action.</span></span>

    <span data-ttu-id="78664-118">Заголовок кредит-ноты продажи содержит некоторую информацию из учтенного счета продажи.</span><span class="sxs-lookup"><span data-stu-id="78664-118">The sales credit memo header contains some information from the posted sales invoice.</span></span> <span data-ttu-id="78664-119">Его можно редактировать, например новой информацией, которая отражает соглашение о возврате.</span><span class="sxs-lookup"><span data-stu-id="78664-119">You can edit this, for example, with new information that reflects the return agreement.</span></span>  
3. <span data-ttu-id="78664-120">Измените информацию в строках в соответствии с соглашением, например укажите количество возвращаемых товаров или суммы к возмещению.</span><span class="sxs-lookup"><span data-stu-id="78664-120">Edit information on the lines according to the agreement, such as the number of returned items or the amount to reimburse.</span></span>
4. <span data-ttu-id="78664-121">Выберите действие **Применить операции**.</span><span class="sxs-lookup"><span data-stu-id="78664-121">Choose the **Apply Entries** action.</span></span>
5. <span data-ttu-id="78664-122">В окне **Применить операции клиента** выберите строку с учтенным документом продажи для которого необходимо применить кредит-ноту продажи, а затем выберите действие **Код применения**.</span><span class="sxs-lookup"><span data-stu-id="78664-122">In the **Apply Customer Entries** window, select the line with the posted sales document that you want to apply the sales credit memo to, and then choose the **Applies-to ID** action.</span></span>

    <span data-ttu-id="78664-123">Идентификатор кредит-ноты продажи отображается в поле **Код применения**.</span><span class="sxs-lookup"><span data-stu-id="78664-123">The identifier of the sales credit memo displays in the **Applies-to ID** field.</span></span>
6. <span data-ttu-id="78664-124">В поле **Сумма для применения** введите сумму, которую требуется применить, если она меньше исходной суммы.</span><span class="sxs-lookup"><span data-stu-id="78664-124">In the **Amount to Apply** field, enter the amount that you want to apply if it's smaller than the original amount.</span></span>  

    <span data-ttu-id="78664-125">В нижней части окна **Применить операции клиента** вы можете просмотреть общую сумму для применения для сторнирования всех соответствующих операций, в частности значение в поле **Сальдо** равняется нулю.</span><span class="sxs-lookup"><span data-stu-id="78664-125">At the bottom of the **Apply Customer Entries** window, you can see the total amount to apply to reverse all involved entries, namely when the value in the **Balance** field is zero.</span></span>

    <span data-ttu-id="78664-128">Создав или изменив строки кредит-ноты продажи и указав одно или несколько применений, можно учесть кредит-ноту продажи.</span><span class="sxs-lookup"><span data-stu-id="78664-128">After you create or edit sales credit memo lines, and the single or multiple applications are specified, you can post the sales credit memo.</span></span>   
8. <span data-ttu-id="78664-129">Выберите действие **Учесть и отправить**.</span><span class="sxs-lookup"><span data-stu-id="78664-129">Choose the **Post and Send** action.</span></span>  

<span data-ttu-id="78664-130">Откроется диалоговое окно **Учет и отправка подтверждения**, содержащее предпочитаемый метод отправки для клиента.</span><span class="sxs-lookup"><span data-stu-id="78664-130">The **Post and Send Confirmation** dialog box opens showing the preferred sending method for the customer.</span></span> <span data-ttu-id="78664-131">Можно изменить метод отправки, нажав кнопку поиска у поля **Кому отправить документ**.</span><span class="sxs-lookup"><span data-stu-id="78664-131">You can change the sending method by choosing the lookup button for the **Send Document** to field.</span></span> <span data-ttu-id="78664-132">Дополнительные сведения см. в разделе [Практическое руководство. Настройка профилей отправки документов](sales-how-setup-document-send-profiles.md).</span><span class="sxs-lookup"><span data-stu-id="78664-132">For more information, see [How to: Set Up Document Sending Profiles](sales-how-setup-document-send-profiles.md).</span></span>  

<span data-ttu-id="78664-133">Учтенные документы продажи, к которым была применена кредит-нота, сторнируются, и можно создать возврат средств для клиента.</span><span class="sxs-lookup"><span data-stu-id="78664-133">The posted sales documents that you applied the credit memo to are now reversed, and a refund payment can be created for the customer.</span></span> <span data-ttu-id="78664-134">Кредит-нота продажи удаляется и заменяется новым документом в списке учтенных кредит-нот продажи.</span><span class="sxs-lookup"><span data-stu-id="78664-134">The sales credit memo is removed and replaced with a new document in the list of posted sales credit memos.</span></span>
