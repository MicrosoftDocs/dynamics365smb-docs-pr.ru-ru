# <a name="how-to-set-up-the-envestnet-yodlee-bank-feeds-service"></a>Практическое руководство. Настройка службы банковских выписок Envestnet Yodlee
Вы можете импортировать электронные банковские выписки для быстрого заполнения окна **Журнал выверки платежей**, чтобы можно было применять платежи и выверять банковский счет. Дополнительные сведения см. в разделе [Автоматическое применение платежей и выверка банковских счетов](receivables-apply-payments-auto-reconcile-bank-accounts.md).

Служба банковских выписок Envestnet Yodlee устанавливается как расширение [!INCLUDE[d365fin](includes/d365fin_md.md)] и готова к включению. Дополнительные сведения см. в разделе [Настройка [!INCLUDE[d365fin](includes/d365fin_md.md)] с помощью расширений](ui-extensions.md).

> [!NOTE]
> Служба банковских выписок поддерживается только в США, Канаде и Великобритании.

После включения службы банковских выписок вы должны связать банковский счет со счетом интернет-банка, с которого поступают выписки. Банковские счета связываются со счетами интернет-банков в следующих различных ситуациях:

* В [!INCLUDE[d365fin](includes/d365fin_md.md)] отсутствует банковский счет для вашего счета интернет-банка. Поэтому вы создаете банковский счет путем связывания из счета интернет-банка.
* В [!INCLUDE[d365fin](includes/d365fin_md.md)] существует банковский счет, которые требуется связать со счетом интернет-банка.
* Связанный банковский счет нужно отвязать, поскольку вы хотите перестать использовать службу банковских выписок для счета.
* Счета интернет-банка изменились и вы хотите обновить информацию о банковских счетах в [!INCLUDE[d365fin](includes/d365fin_md.md)].

Если служба банковских выписок включена, вы можете настроить банковский счет на автоматический импорт новых банковских выписок в окно **Журнал выверки платежей** каждые два часа. Транзакции по платежам, которые уже были учтены как примененные или выверенные в окне **Журнал выверки платежей**, не будут импортированы. Дополнительные сведения см. в разделе “Включение автоматического импорта банковских выписок”.

> [!NOTE]  
>   Если вы используете сопровождаемую настройку организации, то некоторые из описанных ниже шагов будут выполнены автоматически при настройке банковского счета организации. Дополнительные сведения см. в разделе [Добро пожаловать в [!INCLUDE[d365fin_long](includes/d365fin_long_md.md)]](index.md).

## <a name="to-enable-the-bank-feed-service"></a>Включение службы банковских выписок
1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Банковские счета**, а затем выберите связанную ссылку.
2. Откройте банковский счет, который будет использоваться для службы банковских выписок.
3. В окне **Банковский счет** в поле **Формат импорта банковской выписки** выберите YODLEEBANKFEED.  

Служба банковских выписок будет включена, когда вы свяжете банковский счет со связанным счетом интернет-банка. Перейти к следующей процедуре.  

## <a name="to-create-a-new-linked-bank-account"></a>Создание нового связанного банковского счета
1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Банковские счета**, а затем выберите связанную ссылку.
2. Выберите соответствующий банковский счет, а затем выберите **Создать новый связанный банковский счет**. Будет открыто окно Связи **Связывание банковского счета**.

    > [!NOTE]  
>   Это окно показывает фактическую веб-страницу службы банковских выписок Envestnet Yodlee. Термины и функции в этом окне могут отличаться от инструкций в этом разделе.  
3. В окне **Связывание интерактивного банковского счета** в области **Связь счета** с помощью функции поиска найдите банк, в котором у вас открыты один или несколько счетов интернет-банка.
4. Выберите название банка. Откроется область **Вход**.
5. Введите имя пользователя и пароль, которые вы используете для входа в интернет-банк, а затем нажмите кнопку **Далее**.  
6. Служба банковских выписок подготовится связать первый счет указанного интернет-банка с новым банковским счетом в [!INCLUDE[d365fin](includes/d365fin_md.md)].

    > [!NOTE]  
>   Если у вас имеется несколько счетов в интернет-банке, вы должны создать дополнительные банковские счета в [!INCLUDE[d365fin](includes/d365fin_md.md)] для них. См. шаги с 8 по 10.  

    После успешного завершения процесса название банка будет отображаться в области **Мои счета** на вкладке **Связан**. Номер в скобках показывать количество связанных счетов интернет-банка.  
7. Нажмите кнопку **ОК**.

    Если вы связываете только один банковский счет, откроется окно **Карточка банковского счета** и отобразится имя счета интернет-банка. В этом случае задача связывания банковского счета завершена. Остается только настроить банковский счет. Дополнительные сведения см. в разделе [Практическое руководство. Настройка банковских счетов](bank-how-setup-bank-accounts.md).

    Если вы связываете несколько счетов интернет-банка, откроется окно **Связывание банковского счета** со списком счетов интернет-банка, которые еще не связаны с банковскими счетами в [!INCLUDE[d365fin](includes/d365fin_md.md)]. В этом случае выполните следующее действие.  
8. В окне **Связывание банковского счета** выберите строку интернет-банка, а затем выберите действие **Связать с новым банковским счетом**.  

    Откроется окно **Карточка банковского счета** и отобразится имя счета интернет-банка.

    Если в [!INCLUDE[d365fin](includes/d365fin_md.md)] уже существует банковский счет, которые требуется связать с дополнительным счетом интернет-банка, выполните следующий шаг.  
9. В окне **Связывание банковского счета** выберите строку интернет-банка, а затем выберите действие **Связать с существующим банковским счетом**.
10. В окне **Список банк. счетов** выберите банковский счет, который вы хотите связать, а затем нажмите кнопку **ОК**.

## <a name="to-link-a-bank-account-to-an-online-bank-account"></a>Связывание банковского счета со счетом интернет-банка
1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Банковские счета**, а затем выберите связанную ссылку.
2. Выберите строку банковского счета, которые не связан со счетом интернет-банка, а затем выберите действие **Связать с интерактивным банковским счетом**. Будет открыто окно **Связывание интерактивного банковского счета** с заполненным названием банка в области **Связь счета**.
3. Выберите название банка. Откроется область **Вход**.
4. Введите имя пользователя и пароль, которые вы используете для входа в интернет-банк, а затем нажмите кнопку **Далее**.  

    Служба банковских выписок подготовится связать ваш банковский счет в [!INCLUDE[d365fin](includes/d365fin_md.md)] с соответствующим счетом интернет-банка.  

    После успешного завершения процесса названия банка будет отображаться в области **Мои счета** на вкладке **Связан**. Если банк имеет более одного банковского счета, будет связан только счет, выбранный на шаге 2.  
5. Нажмите кнопку **ОК**.

В окне **Список банковских счетов** будет установлен флажок **Связан**.

## <a name="to-unlink-a-bank-account"></a>Отмена связи банковского счета
1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Банковские счета**, а затем выберите связанную ссылку.  
2. Выберите строку связанного банковского счета, который требуется отвязать от счета интернет-банка, а затем выберите действие **Отменить связывание интерактивного банковского счета**.

> [!NOTE]  
>   Если выбрать в диалоговом окне подтверждения **Да**, связь со счетом интернет-банка будет удалена, а реквизиты входа очищены. Чтобы снова связать банковский счет со счетом интернет-банка, необходимо снова войти в интернет-банк. Дополнительные сведения см. в разделе “Связывание банковского счета со счетом интернет-банка“.

## <a name="to-update-bank-account-linking"></a>Обновление связи банковских счетов
1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Банковские счета**, а затем выберите связанную ссылку.
2. Выберите соответствующий банковский счет, а затем выберите действие **Обновить связывание банковских счетов**.

Если существуют проблемы с любым из связанных банковских счетов в окне **Список банковских счетов**, откроется окно **Связывание банковского счета**, указывающее, с каким из банковских счетов имеются проблемы. Проблемы можно разрешить, удалив связь со счетом интернет-банка, а затем заново создав ее. Дополнительные сведения см. в разделе “Связывание банковского счета со счетом интернет-банка“.

## <a name="to-enable-automatic-import-of-bank-statements"></a>Включение автоматического импорта банковских выписок
1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Банковские счета**, а затем выберите связанную ссылку.
2. Выберите строку связанного банковского счета, а затем выберите действие **Настройка автоматического импорта банковских выписок**.
3. В окне **Настройка автоматического импорта банковских выписок** в поле **Включаемое количество дней** укажите, за сколько дней вы хотите получать информацию о банковских транзакциях.

    > [!NOTE]  
>   Рекомендуется установить для этого параметра 7 дней или более.  
4. Установите флажок **Включено**.  

Каждый час в окне **Журнал выверки платежей** будут отображаться новые платежи, прошедшие через счет интернет-банка.

> [!NOTE]  
>   Транзакции по платежам, которые уже были учтены как примененные или выверенные в окне **Журнал выверки платежей**, не будут импортированы.
