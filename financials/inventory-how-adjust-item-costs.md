# <a name="how-to-adjust-item-costs-manually"></a><span data-ttu-id="e0ba9-103">Практическое руководство. Корректировка себестоимости товара вручную</span><span class="sxs-lookup"><span data-stu-id="e0ba9-103">How to: Adjust Item Costs Manually</span></span>
<span data-ttu-id="e0ba9-104">Себестоимость товара (инвентарная стоимость), который вы покупаете, а позже продаете, может изменяться на протяжении его срока службы, например в результате добавления себестоимости перевозки к себестоимости покупки после продажи товара.</span><span class="sxs-lookup"><span data-stu-id="e0ba9-104">The cost of an item (inventory value) that you purchase and later sell may change during its lifetime, for example because a freight cost is added to its purchase cost after you have sold the item.</span></span> <span data-ttu-id="e0ba9-105">Корректировка себестоимости особенно важна в ситуациях, когда товары продаются до того, как на их покупку выставляется счет.</span><span class="sxs-lookup"><span data-stu-id="e0ba9-105">Cost adjustment is especially relevant in situations where you sell goods before you invoice the purchase of those goods.</span></span> <span data-ttu-id="e0ba9-106">Чтобы всегда знать правильную инвентарную стоимость, необходимо регулярно корректировать себестоимость товара.</span><span class="sxs-lookup"><span data-stu-id="e0ba9-106">To always know the correct inventory value, item costs must therefore regularly be adjusted.</span></span> <span data-ttu-id="e0ba9-107">Это гарантирует, что статистика продаж и прибыли всегда актуальна и финансовые KPI верны.</span><span class="sxs-lookup"><span data-stu-id="e0ba9-107">This ensures that sales and profit statistics are up to date and that financial KPIs are correct.</span></span>

<span data-ttu-id="e0ba9-108">В [!INCLUDE[d365fin](includes/d365fin_md.md)] себестоимость товаров автоматически корректируется при каждой складской транзакции, например при учете счета покупки для товара.</span><span class="sxs-lookup"><span data-stu-id="e0ba9-108">In [!INCLUDE[d365fin](includes/d365fin_md.md)], item costs are automatically adjusted every time that an inventory transaction occurs, such as when posting a purchase invoice for an item.</span></span>

<span data-ttu-id="e0ba9-109">Можно также использовать функцию для корректировки себестоимости одного или нескольких товаров вручную.</span><span class="sxs-lookup"><span data-stu-id="e0ba9-109">You can also use a function to manually adjust the costs of one or more items.</span></span> <span data-ttu-id="e0ba9-110">Это полезно, например, если известно, что себестоимость товара изменилась по причинам, отличным от товарной транзакции.</span><span class="sxs-lookup"><span data-stu-id="e0ba9-110">This is useful, for example, when you know that item costs have changed for other reasons than item transactions.</span></span>

<span data-ttu-id="e0ba9-111">Себестоимость товаров корректируется методом FIFO или средней себестоимости в зависимости от параметров, выбранных при сопровождаемой настройке **Настройка организации**.</span><span class="sxs-lookup"><span data-stu-id="e0ba9-111">Item costs are adjusted by the FIFO or Average costing method, depending on your selection in the **Set Up My Company** assisted setup.</span></span> <span data-ttu-id="e0ba9-112">Дополнительные сведения см. в разделе [Подготовьтесь к ведению бизнеса](ui-get-ready-business.md).</span><span class="sxs-lookup"><span data-stu-id="e0ba9-112">For more information, see [Getting Ready for Doing Business](ui-get-ready-business.md).</span></span>  

<span data-ttu-id="e0ba9-113">Если используется метод учета себестоимости FIFO, то себестоимость единицы товара — это фактическая стоимость товара в любой приходной накладной.</span><span class="sxs-lookup"><span data-stu-id="e0ba9-113">If you use the FIFO costing method, then an item’s unit cost is the actual value of any receipt of the item.</span></span> <span data-ttu-id="e0ba9-114">Запасы оцениваются из предположения, что товары, помещенные на склад первыми, продаются первыми.</span><span class="sxs-lookup"><span data-stu-id="e0ba9-114">Inventory is valuated with the assumption that the first items placed in inventory are sold first.</span></span>

<span data-ttu-id="e0ba9-115">Если используется метод средней себестоимости, себестоимость единицы товара рассчитывается как средняя себестоимость единицы в каждый момент времени после покупки.</span><span class="sxs-lookup"><span data-stu-id="e0ba9-115">If you use the Average costing method, then an item’s unit cost is calculated as the average unit cost at each point in time after a purchase.</span></span> <span data-ttu-id="e0ba9-116">Запасы переоцениваются из допущения, что все запасы продаются одновременно.</span><span class="sxs-lookup"><span data-stu-id="e0ba9-116">Inventory is valuated with the assumption that all inventories are sold simultaneously.</span></span>

<span data-ttu-id="e0ba9-117">Функция корректировки себестоимости обрабатывает только операции стоимости, которые еще не были скорректированы.</span><span class="sxs-lookup"><span data-stu-id="e0ba9-117">The cost adjustment function processes only value entries that have not yet been adjusted.</span></span> <span data-ttu-id="e0ba9-118">Если функция выполняется в ситуации, в которой измененные входящие значения себестоимости необходимо направить в соответствующие выходные операции, для выполнения этой задачи создаются новые корректирующие операции стоимости, в основе которых лежат данные исходных операций стоимости, хотя сами они содержат сумму коррекции.</span><span class="sxs-lookup"><span data-stu-id="e0ba9-118">If the function encounters a situation where changed inbound costs need to be forwarded to associated outbound entries, then new adjustment value entries are created, which are based on the information in the original value entries but contain the adjustment amount.</span></span> <span data-ttu-id="e0ba9-119">Функция корректировки себестоимости использует дату учета исходной операции стоимости в операции коррекции, если дата не попадает в закрытый период учета запасов.</span><span class="sxs-lookup"><span data-stu-id="e0ba9-119">The cost adjustment function uses the posting date of the original value entry in the adjustment entry, unless that date is in a closed inventory period.</span></span> <span data-ttu-id="e0ba9-120">В этом случае программа использует начальную дату следующего открытого периода учета запасов.</span><span class="sxs-lookup"><span data-stu-id="e0ba9-120">In that case, the program uses the starting date of the next open inventory period.</span></span> <span data-ttu-id="e0ba9-121">Если периоды учета запасов не используются, дата в поле **Разрешить учет с** в окне **Главная книга - настройка** будет определяться при учете операции корректировки.</span><span class="sxs-lookup"><span data-stu-id="e0ba9-121">If inventory periods are not used, then the date in the **Allow Posting From** field in the **General Ledger Setup** window will define when the adjustment entry is posted.</span></span>

<span data-ttu-id="e0ba9-122">Изменения в стоимости товара в результате торговых операций автоматически выверяются с финансовыми книгами при учете товарных транзакций.</span><span class="sxs-lookup"><span data-stu-id="e0ba9-122">Changes in inventory value from trade are automatically reconciled with your financial books when you post item transactions.</span></span> <span data-ttu-id="e0ba9-123">Дополнительные сведения см. в подразделе "Выверка запасов" раздела [Запасы](inventory-manage-inventory.md).</span><span class="sxs-lookup"><span data-stu-id="e0ba9-123">For more information, see the "Inventory Reconciliation" section in [Inventory](inventory-manage-inventory.md).</span></span>
