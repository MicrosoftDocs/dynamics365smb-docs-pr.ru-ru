<span data-ttu-id="ad243-105">Для отгрузки нескладируемого товара конкретному клиенту можно создать специальный заказ.</span><span class="sxs-lookup"><span data-stu-id="ad243-105">You can create a special order for a specific nonstock item to be shipped to a specific customer.</span></span> <span data-ttu-id="ad243-106">Поставщик отгружает товар на склад организации, после чего этот товар отгружается в адрес клиента либо отдельно, либо вместе с другими товарами по другому заказу.</span><span class="sxs-lookup"><span data-stu-id="ad243-106">Your vendor ships the item to your warehouse and you can then ship the item on to your customer either independently or together with other items on another order.</span></span>  

<span data-ttu-id="ad243-107">Специальные заказы предполагают наличие связи между заказом на покупку и заказом на продажу, позволяющей обеспечить подбор конкретного нескладируемого товара и его доставку клиенту.</span><span class="sxs-lookup"><span data-stu-id="ad243-107">Special orders imply that the purchase and sales order are linked to ensure that the specific nonstock item is picked and delivered to the customer.</span></span>  

<span data-ttu-id="ad243-108">Перед использованием этой функции следует сначала настроить карточки клиента, поставщика и товара, которые необходимы для создания заказа.</span><span class="sxs-lookup"><span data-stu-id="ad243-108">Before you can use this feature, you must first set up the customer, vendor, and item cards necessary for the order.</span></span>  

## <a name="to-create-a-special-order"></a><span data-ttu-id="ad243-109">Создание специального заказа</span><span class="sxs-lookup"><span data-stu-id="ad243-109">To create a special order</span></span>  
1.  <span data-ttu-id="ad243-110">Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Заказ на продажу**, а затем выберите связанную ссылку.</span><span class="sxs-lookup"><span data-stu-id="ad243-110">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Sales Order**, and then choose the related link.</span></span>  
2. <span data-ttu-id="ad243-111">Выберите действие **Создать**.</span><span class="sxs-lookup"><span data-stu-id="ad243-111">Choose the **New** action.</span></span> <span data-ttu-id="ad243-112">Создайте и заполните заказ на продажу товара.</span><span class="sxs-lookup"><span data-stu-id="ad243-112">Create and fill in a  sales order for the item.</span></span> <span data-ttu-id="ad243-113">Дополнительные сведения см. в разделе [Практическое руководство. Продажа продукции](sales-how-sell-products.md).</span><span class="sxs-lookup"><span data-stu-id="ad243-113">For more information, see [How to: Sell Products](sales-how-sell-products.md).</span></span>
3.  <span data-ttu-id="ad243-114">На экспресс-вкладке **Строки** заполните строку продажи.</span><span class="sxs-lookup"><span data-stu-id="ad243-114">On the **Lines** FastTab, fill in the sales line.</span></span> <span data-ttu-id="ad243-115">В поле **Код закупки** выберите код покупки, для которого выбрано поле **Специальный заказ**.</span><span class="sxs-lookup"><span data-stu-id="ad243-115">In the **Purchasing Code** field, select a purchasing code that has the **Special Order** field selected.</span></span>

    <span data-ttu-id="ad243-116">Теперь необходимо создать заказ на покупку на основе журнала заявок.</span><span class="sxs-lookup"><span data-stu-id="ad243-116">You must now create a purchase order from a requisition worksheet.</span></span>  
4. <span data-ttu-id="ad243-117">Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Журнал заявок**, затем выберите связанную ссылку.</span><span class="sxs-lookup"><span data-stu-id="ad243-117">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Requisition Worksheet**, and then choose the related link.</span></span>  
5. <span data-ttu-id="ad243-118">Выберите действие **Специальный заказ**, затем выберите действие **Получить заказы на продажу**.</span><span class="sxs-lookup"><span data-stu-id="ad243-118">Choose the **Special Order** action, and then choose the **Get Sales Orders** action.</span></span>  
6.  <span data-ttu-id="ad243-119">В окне **Получить заказы на продажу** отображаются результаты, в которых **Номер документа** является номером заказа на продажу.</span><span class="sxs-lookup"><span data-stu-id="ad243-119">In the **Get Sales Orders** window, show results where the **Document No.** is the sales order number.</span></span> <span data-ttu-id="ad243-120">Нажмите кнопку **ОК**.</span><span class="sxs-lookup"><span data-stu-id="ad243-120">Choose the **OK** button.</span></span> <span data-ttu-id="ad243-121">Для товара создается строка журнала заявок.</span><span class="sxs-lookup"><span data-stu-id="ad243-121">A requisition worksheet line is created for the item.</span></span>  
7.  <span data-ttu-id="ad243-122">В строке заявки выберите в поле **Указание** выберите **Новое**.</span><span class="sxs-lookup"><span data-stu-id="ad243-122">On the requisition worksheet line, in the **Action Message** field, select **New**.</span></span>  
8.  <span data-ttu-id="ad243-123">В окне **Журнал заявок** выберите действие **Выполнить указание**.</span><span class="sxs-lookup"><span data-stu-id="ad243-123">In the **Req. Worksheet** window, choose the **Carry Out Action Message** action.</span></span> <span data-ttu-id="ad243-124">Открывается окно **Выполнение указаний - заявка**.</span><span class="sxs-lookup"><span data-stu-id="ad243-124">The **Carry Out Action Msg. - Req.** window opens.</span></span> <span data-ttu-id="ad243-125">Нажмите кнопку **ОК**.</span><span class="sxs-lookup"><span data-stu-id="ad243-125">Choose the **OK** button.</span></span>  

    <span data-ttu-id="ad243-126">Отобразится сообщение о том, что созданы заказы на покупку.</span><span class="sxs-lookup"><span data-stu-id="ad243-126">A message appears telling you that the purchase orders have been created.</span></span> <span data-ttu-id="ad243-127">Нажмите кнопку **OK**.</span><span class="sxs-lookup"><span data-stu-id="ad243-127">Choost the **OK** button.</span></span>  

<span data-ttu-id="ad243-128">Заказ на покупку, созданный как специальный заказ для заказа на продажу, учитывается системой планирования, поскольку балансирует спрос и предложение.</span><span class="sxs-lookup"><span data-stu-id="ad243-128">A purchase order created as a special order for a sales order is respected by the planning system as it balances demand and supply.</span></span> <span data-ttu-id="ad243-129">Это означает, что заказ на покупку (предложение) остается связанным с заказом на продажу (требованием), даже если заказ на покупку мог бы быть использован для поставки по более раннему требованию.</span><span class="sxs-lookup"><span data-stu-id="ad243-129">That is, the purchase order (supply) remains linked to the sales order (demand), even if that purchase order could supply another earlier demand.</span></span> <span data-ttu-id="ad243-130">Дополнительные сведения см. в разделе [Сведения о проектировании. Политики дозаказа](design-details-reservation-order-tracking-and-action-messaging.md).</span><span class="sxs-lookup"><span data-stu-id="ad243-130">For more information, see [Design Details: Reordering Policies](design-details-reservation-order-tracking-and-action-messaging.md).</span></span>

>  <span data-ttu-id="ad243-131">Нельзя использовать функцию специального заказа, если товар уже зарезервирован.</span><span class="sxs-lookup"><span data-stu-id="ad243-131">You cannot use the special order functionality if the item is already reserved.</span></span> <span data-ttu-id="ad243-132">Поэтому для товаров, которые продаются по специальным заказам, убедитесь, что значение в поле **Резервировать** в карточке товара не равно **Всегда**.</span><span class="sxs-lookup"><span data-stu-id="ad243-132">Therefore, for items that are sold on special orders, make sure the **Reserve** field on the item card is not set to **Always**.</span></span>
