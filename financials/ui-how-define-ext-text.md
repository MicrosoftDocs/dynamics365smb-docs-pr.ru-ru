1. <span data-ttu-id="28880-107">Откройте карточку для товара, для которого необходимо добавить расширенный текст, затем выберите действие **Расширенный текст**.</span><span class="sxs-lookup"><span data-stu-id="28880-107">Open the card for an item that you want to add extended text to, and then choose the **Extended Text** action.</span></span>
