# <a name="preview-posting-results"></a><span data-ttu-id="4f4e2-103">Предварительный просмотр результатов учета</span><span class="sxs-lookup"><span data-stu-id="4f4e2-103">Preview Posting Results</span></span>
<span data-ttu-id="4f4e2-104">Во всех документах и журналах, которые можно учитывать, можно нажать кнопку **Предварительный просмотр учета** для просмотра различных типов операций, которые будут созданы при учете документа или журнала.</span><span class="sxs-lookup"><span data-stu-id="4f4e2-104">On every document and journal that can be posted, you can choose the **Preview Posting** button to review the different types of entries that will be created when you post the document or journal.</span></span>

## <a name="to-preview-gl-entries-that-will-result-from-posting-a-purchase-invoice"></a><span data-ttu-id="4f4e2-105">Просмотр операций ГК, которые появятся в результате учета счета покупки</span><span class="sxs-lookup"><span data-stu-id="4f4e2-105">To preview G/L entries that will result from posting a purchase invoice</span></span>
1. <span data-ttu-id="4f4e2-106">Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Счета покупки**, а затем выберите связанную ссылку.</span><span class="sxs-lookup"><span data-stu-id="4f4e2-106">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Purchase Invoices**, and then choose the related link.</span></span>
2. <span data-ttu-id="4f4e2-107">Создайте счет покупки.</span><span class="sxs-lookup"><span data-stu-id="4f4e2-107">Create a purchase invoice.</span></span> <span data-ttu-id="4f4e2-108">Дополнительные сведения см. в разделе [Регистрация покупок](purchasing-how-record-purchases.md).</span><span class="sxs-lookup"><span data-stu-id="4f4e2-108">For more information, see [Record Purchases](purchasing-how-record-purchases.md).</span></span>
3. <span data-ttu-id="4f4e2-109">Выберите **Предварительный просмотр учета**.</span><span class="sxs-lookup"><span data-stu-id="4f4e2-109">Choose **Preview Posting**.</span></span>
4. <span data-ttu-id="4f4e2-110">В окне **Предварительный просмотр учета** выберите **Операция ГК**, а затем выберите **Показать связанные операции**.</span><span class="sxs-lookup"><span data-stu-id="4f4e2-110">In the **Posting Preview** window, select **G/L Entry**, and then choose **Show Related Entries**.</span></span>  
   <span data-ttu-id="4f4e2-111">В окне **Предварительный просмотр операций ГК** будут показаны операции, которые будут созданы в результате учета счета покупки.</span><span class="sxs-lookup"><span data-stu-id="4f4e2-111">The **G/L Entries Preview** window shows which entries will be created when you post the purchase invoice.</span></span>
