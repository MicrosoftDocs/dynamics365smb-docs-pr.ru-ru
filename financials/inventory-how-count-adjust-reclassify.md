# <a name="how-to-count-adjust-and-reclassify-inventory"></a>Практическое руководство. Подсчет, корректировка и повторная классификация запасов
В каждом финансовом году по крайней мере один раз вы должны провести инвентаризацию запасов, т. е. пересчитать все товары в запасах, чтобы определить, совпадает ли количество, зарегистрированное в базе данных, с фактическим количеством товаров на складе. Когда фактическое количество известно, его необходимо учесть в главной книге в рамках оценки стоимости запасов в конце периода.

Несмотря на то, что подсчет товаров в наличии производится не реже одного раза в год, некоторые товары могут подсчитываться более часто, так как являются наиболее ценными или часто перемещаются и составляют основную часть бизнеса. Для этого можно назначить специальные периоды подсчета для этих товаров. Дополнительные сведения см. в подразделе "Выполнение циклического подсчета".

Если необходимо скорректировать записанное количество запасов в связи с пересчетом или по другим причинам, с помощью журнала товаров можно изменить книгу операций с запасами непосредственно без учета бизнес-транзакций. Кроме того, можно корректировать отдельный товар или карточку товара.

Если необходимо изменить атрибуты операций книги товаров, а также количество, для этого можно использовать журнал реклассификации товаров. Типичные атрибуты для реклассификации — это серийные номера или номера партий, даты истечения срока и измерения.

## <a name="to-perform-a-physical-inventory"></a>Выполнение физической инвентаризации
Для проверки соответствия зарегистрированного в программе количества физическому количеству товара на складе необходимо выполнять инвентаризацию (т.е. производить подсчет фактических наличных товаров) в конце финансового года или даже чаще. При наличии разниц необходимо учесть их в счетах товаров перед выполнением оценки стоимости товара на складе.

Помимо задачи физического подсчета, процесс завершения включает следующие три задачи:

- Вычисление ожидаемых запасов.
- Печать отчета, используемого при подсчете.
- Введите фактические рассчитанные запасы и выполните их учет.

### <a name="to-calculate-the-expected-inventory"></a>Вычисление ожидаемых запасов
1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Журналы инвентаризации**, а затем выберите связанную ссылку.
2. Выберите действие **Расчет запасов**.
3. В окне **Расчет запасов** укажите условия создания строк журналов, например включаются ли товары с нулевой величиной складского запаса.
4. Установите фильтры, чтобы выполнить расчет запасов только для определенных товаров, ячеек складов или измерений.
5. Нажмите кнопку **ОК**.

> [!NOTE]  
>   Товарные операции обрабатываются согласно указанной информации, а строки создаются в журнале инвентаризации. Обратите внимание, что поле **Кол-во (инвентаризация)** автоматически заполняется тем же количеством, что и поле **Расчетное кол-во**. Благодаря этой функции пропадает необходимость вводить фактический рассчитанный запас, когда он совпадает с расчетным количеством товаров. Если сосчитанное количество отличается от уже введенного, в поле **Расчетное кол-во**, необходимо перезаписывать реальное сосчитанное количество.

### <a name="to-print-the-report-used-when-counting"></a>Печать отчета, используемого при расчете
1. В окне **Журнал инвентаризации**, содержащем рассчитанные ожидаемые запасы, выберите **Печать**.
2. В окне **Список инвентаризации** укажите, должно ли в отчете отображается расчетное количество и должен ли отчет содержать список товаров согласно серийным номерами/номерам партий.
3. Установите фильтры, чтобы выполнить печать отчета только для определенных товаров, ячеек складов или измерений.
4. Нажмите кнопку **Печать**.

Теперь сотрудники могут перейти к подсчету запасов и фиксации расхождений в печатном отчете.

### <a name="to-enter-and-post-the-actual-counted-inventory"></a>Ввод и учет фактических рассчитанных запасов
1. В каждой строке в окне **Журнал инвентаризации**, в которой фактическое количество доступного товара, определенное путем физического подсчета, отличается от расчетного количества, введите фактическое количество доступного товара в поле **Кол-во (инвентаризация)**.

    Связанные поля обновляются соответствующим образом.

    > [!NOTE]  
>   Если при физическом расчете выявлены разницы, возникшие в результате выполнения учета товаров с ошибочными кодами склада, не вводите эти разницы в журнал инвентаризации. Вместо этого, используйте журнал реклассификации или заказ на перемещение для перемещения товаров в нужные склады. Дополнительные сведения см. в разделе "Журнал реклассификации" или "Практическое руководство. Создание заказов на перемещение".

2. Чтобы скорректировать рассчитанное количество до фактического подсчитанного количества, выберите действие **Учет**.

    Создаются как операции книги товаров, так и операции книги инвентаризации. Откройте карточку товара для просмотра окончательных операций инвентаризации.

3. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Товары**, а затем выберите связанную ссылку.
4. Чтобы проверить пересчет запасов, откройте нужную карточку товара и выберите **Книга операций по инвентаризации**.

# <a name="to-perform-cycle-counting"></a>Выполнение цикличного пересчета
Несмотря на то что подсчет товаров в наличии производится не реже одного раза в год, некоторые товары могут подсчитываться более часто, так как являются наиболее ценными или часто перемещаются и составляют основную часть бизнеса. Для этого можно назначить специальные периоды подсчета для этих товаров.

> [!NOTE]  
>   Если на складе настроен расширенный подбор и размещение, следует сначала использовать окно **Журнал инвентаризации**, а затем можно использовать функцию **Расчет коррекции склада** в окне **Журнал инвентаризации**.

## <a name="to-set-up-counting-periods"></a>Настройка периодов пересчета
Инвентаризация обычно проводится через определенные интервалы времени, например, раз в месяц, квартал или год. Можно настроить любую периодичность проведения инвентаризации.

Сначала нужно настроить периодичность проведения инвентаризации, которая затем присваивается каждому товару. При выполнении инвентаризацию и использовании функции **Расчет периода подсчета** в журнале инвентаризации строки для товаров создаются автоматически.

1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Периоды проведения инвентаризации**, а затем выберите связанную ссылку.  
2. Заполните соответствующим образом поля. [!INCLUDE[tooltip-inline-tip](includes/tooltip-inline-tip_md.md)]

## <a name="to-assign-a-counting-period-to-an-item"></a>Назначение периода пересчета товару  
1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Товары**, а затем выберите связанную ссылку.  
2. Выберите товар, которому необходимо присвоить период пересчета.  
3. В поле **Код инвентарной описи** выберите соответствующий период пересчета  
4. Нажмите кнопку **Да**, чтобы изменить код и вычислить первый период пересчета для товара. При следующем расчете периода пересчета в журнале инвентаризации товар отображается в виде строки в окне **Выбор товара для инвентаризации**. Теперь можно приступить к периодическому подсчету товара.

## <a name="to-initiate-a-count-based-on-counting-periods"></a>Запуск пересчета на основе периодов пересчета
1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Журнал инвентаризации**, а затем выберите связанную ссылку.
2. Выберите действие **Расчет периода подсчета**.

    Откроется окно **Инвентаризация - выбор товара**, в котором содержатся товары, для которых был установлен период пересчета и которые необходимо пересчитать в соответствии с периодом пересчета.
3. Выполните физическую инвентаризацию. Дополнительные сведения см. в подразделе "Выполнение физической инвентаризации".

## <a name="to-adjust-the-inventory-of-one-item"></a>Корректировка запасов одного товара
После физического подсчета товаров в области склада можно воспользоваться функцией **Корректировать запасы** для регистрации фактического количества товаров.

1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Товары**, а затем выберите связанную ссылку.
2. Выберите товар, для которого требуется коррекция запасов, а затем выберите действие **Корректировать запасы**.
3. В поле **Новые запасы** введите количество запасов, которое требуется включить в запись для товара.
4. Нажмите кнопку **ОК**.

Запасы товара скорректированы. Новое количество отображается в поле **Текущие запасы** в окне **Корректировка запасов** и в поле **Запасы** в окне **Карточка товара**.

Функция **Корректировать запасы** также позволяет легко и удобно помещать приобретенные товары в запасы, если для записи покупок не используются заказы или счета покупки. Дополнительные сведения см. в разделе [Практическое руководство. Регистрация покупок](purchasing-how-record-purchases.md).

> [!NOTE]  
>   После корректировки запасов необходимо обновить запасы с использованием текущей рассчитанной стоимости. Дополнительные сведения см. в разделе [Практическое руководство. Переоценка запасов](inventory-how-revalue-inventory.md).

## <a name="to-adjust-the-inventory-quantity-of-one-or-more-items"></a>Корректировка количество запасов для одного или нескольких товаров.
В окне **Журнал товаров** можно учитывать транзакции напрямую для корректировки запасов в связи с покупками, продажами, положительными и отрицательными корректировками без использования или документов.

Если журнал товаров часто используется для учета одинаковых или похожих строк журнала, например, в связи с расходом материалов, для упрощения повторяющейся работы можно использовать окно **Стандартный журнал товаров**. Дополнительные сведения см в подразделе "Стандартные журналы" раздела [Работа с финансовыми журналами](ui-work-general-journals.md).

1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Журналы товаров**, а затем выберите связанную ссылку.
2. Заполните соответствующим образом поля. [!INCLUDE[tooltip-inline-tip](includes/tooltip-inline-tip_md.md)]
3. Выберите действие **Учет**, чтобы выполнить корректировки товаров.

> [!NOTE]  
>   После корректировки запасов необходимо обновить запасы с использованием текущей рассчитанной стоимости. Дополнительные сведения см. в разделе [Практическое руководство. Переоценка запасов](inventory-how-revalue-inventory.md).

## <a name="to-reclassify-an-items-lot-number"></a>Реклассификация номер партии товара
1. Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Журналы реклассификации товаров**, а затем выберите связанную ссылку.
2. В окне **Журнал реклассификации товаров** заполните требуемые поля.
3. В поле **Номер партии** введите текущий номер партии товаров.
4. В поле **Новый номер партии** введите новый номер партии товаров.
5. Выберите действие **Учесть**.
