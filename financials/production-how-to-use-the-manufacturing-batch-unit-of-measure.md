# <a name="work-with-manufacturing-batch-units-of-measure"></a><span data-ttu-id="7e4bd-104">Работа с единицами измерения производственной партии</span><span class="sxs-lookup"><span data-stu-id="7e4bd-104">Work with Manufacturing Batch Units of Measure</span></span>
<span data-ttu-id="7e4bd-105">Если при хранении товара используется одна единица измерения, а при производстве — другая, может быть создан производственный заказ, в котором единица измерения производственной партии используется для расчета правильного количества компонентов при выполнении пакетного задания **Обновление произв. заказа**.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-105">If an item is stocked in one unit of measure but produced in another, a production order is created that uses a manufacturing batch unit of measure to calculate the correct quantity of the components during the **Refresh Production Order** batch job.</span></span> <span data-ttu-id="7e4bd-106">Примером расчета единицы измерения производственной партии является ситуация, когда хранение товара на складе осуществляется поштучно, а производство — в тоннах.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-106">An example of a manufacturing batch unit of measure calculation is when a manufactured item is stocked in pieces but produced in tons.</span></span>  

## <a name="to-create-a-production-bom-using-a-batch-unit-of-measure"></a><span data-ttu-id="7e4bd-107">Создание производственной спецификации с использованием единицы измерения партии</span><span class="sxs-lookup"><span data-stu-id="7e4bd-107">To create a production BOM using a batch unit of measure</span></span>  
1.  <span data-ttu-id="7e4bd-108">Единица измерения производственной партии настраивается как альтернативная единица измерения в окне **Товар - единицы измерения** для производимого товара.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-108">The manufacturing batch unit of measure is set up as an alternative unit of measure in the **Item Units of Measure** window on the item to be produced.</span></span> <span data-ttu-id="7e4bd-109">Единица измерения для партии не заменяет базовую единицу измерения товара.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-109">The batch unit of measure will not replace the base unit of measure on the item.</span></span>  
2.  <span data-ttu-id="7e4bd-110">Создайте производственную спецификацию для товара, настроенного с использованием единицы измерения производственной партии.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-110">Create a production BOM for the item set up with the manufacturing batch unit of measure.</span></span> <span data-ttu-id="7e4bd-111">Дополнительные сведения см. в разделе [Создание производственных спецификаций](production-how-to-create-production-boms.md).</span><span class="sxs-lookup"><span data-stu-id="7e4bd-111">For more information, see [Create Production BOMs](production-how-to-create-production-boms.md).</span></span>  
3.  <span data-ttu-id="7e4bd-112">В поле **Код единицы измерения** выберите единицу измерения производственной партии.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-112">In the **Unit of Measure Code** field, select the manufacturing batch unit of measure.</span></span>  
4.  <span data-ttu-id="7e4bd-113">Для каждой строки производственной спецификации в поле **Количество в** введите количество товаров-компонентов, необходимое для создания данной единицы измерения производственной партии.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-113">For each production BOM line, in the **Quantity Per** field, enter the quantity of this component item that is required to create this batch unit of measure.</span></span>  
5.  <span data-ttu-id="7e4bd-114">Откройте окно **Карточка товара** для соответствующего товара.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-114">Open the **Item Card** for the related item.</span></span>  

    <span data-ttu-id="7e4bd-115">На экспресс-вкладке **Пополнение** в поле **Номер производственной спецификации** выберите созданную ранее производственную спецификацию.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-115">On the **Replenishment** FastTab, in the **Production BOM No.** field, select the production BOM created above.</span></span>  
6.  <span data-ttu-id="7e4bd-116">Создайте заголовок производственного заказа, используя настройку товара с единицами измерения производственной партии.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-116">Create a production order header using the item set up with the manufacturing batch unit of measure.</span></span> <span data-ttu-id="7e4bd-117">Дополнительные сведения см. в разделе [Создание производственных заказов](production-how-to-create-production-orders.md).</span><span class="sxs-lookup"><span data-stu-id="7e4bd-117">For more information, see [Create Production Orders](production-how-to-create-production-orders.md).</span></span>  
7.  <span data-ttu-id="7e4bd-118">Выберите действие **Обновить**, затем выберите кнопку **ОК**.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-118">Choose the **Refresh** action, and then choose  the **OK** button.</span></span>  

<span data-ttu-id="7e4bd-119">Для просмотра результатов перейдите на экспресс-вкладку **Строки**, выберите действие **Строка**, затем выберите действие **Компоненты**.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-119">On the **Lines** FastTab, choose the **Line** action, and then choose the **Components** action to view the result.</span></span> <span data-ttu-id="7e4bd-120">Программа вычислит правильное количество компонентов, необходимое для соблюдения производственной спецификации, используя единицу измерения производственной партии.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-120">The program calculates the correct quantity of the components needed to satisfy the production BOM based on the manufacturing batch unit of measure.</span></span>  

## <a name="to-calculate-a-manufacturing-batch-unit-of-measure-on-a-production-order"></a><span data-ttu-id="7e4bd-121">Вычисление единицы измерения производственной партии для производственного заказа</span><span class="sxs-lookup"><span data-stu-id="7e4bd-121">To calculate a manufacturing batch unit of measure on a production order</span></span>  
1.  <span data-ttu-id="7e4bd-122">Создайте заголовок производственного заказа, используя настройку товара с единицами измерения производственной партии.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-122">Create a production order header using the item set up with the manufacturing batch unit of measure.</span></span>

3.  <span data-ttu-id="7e4bd-124">В поле **Кол-во** введите то же количество, которое указано в заголовке.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-124">In the **Quantity** field, enter the same quantity used in the header.</span></span>  
4.  <span data-ttu-id="7e4bd-125">В поле **Код единицы измерения** выберите код единицы измерения производственной партии.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-125">In the **Unit of Measure Code** field, select the manufacturing batch unit of measure code.</span></span>  
5.  <span data-ttu-id="7e4bd-126">Выберите действие **Обновить**.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-126">Choose the **Refresh** action.</span></span>
6.  <span data-ttu-id="7e4bd-127">На экспресс-вкладке **Вычислить** снимите флажок **Строки**.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-127">On the **Calculate** FastTab, clear the **Lines** check box.</span></span>  
7.  <span data-ttu-id="7e4bd-128">Нажмите кнопку **ОК**.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-128">Choose the **OK** button.</span></span>  
8.  <span data-ttu-id="7e4bd-129">Для просмотра результатов перейдите на экспресс-вкладку **Строки**, выберите действие **Строка**, затем выберите действие **Компоненты**.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-129">On the **Lines** FastTab, choose the **Line** action, and then choose the **Components** action to view the result.</span></span> <span data-ttu-id="7e4bd-130">Правильное количество компонентов, необходимое для соблюдения производственной спецификации, вычисляется с помощью единицы измерения производственной партии.</span><span class="sxs-lookup"><span data-stu-id="7e4bd-130">The correct quantity of the components needed to satisfy the production BOM is calculated based on the manufacturing batch unit of measure.</span></span>
