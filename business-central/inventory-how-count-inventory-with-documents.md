# <a name="count-and-adjust-inventory-using-documents"></a>Подсчет и корректировка запасов с использованием документов

Можно выполнять физическую инвентаризацию товаров с помощью документов заказа инвентаризации и записи инвентаризации. Страница **Заказ инвентаризации** используется для организации проекта полной инвентаризации, например по каждому складу. Страница **Запись инвентаризации** используется для фиксации и дальнейшей передачи фактических количеств товаров. Можно создать несколько записей для одного заказа, например чтобы распределить группы товаров по различным сотрудникам.

Из каждой записи можно распечатать отчет **Запись инвентаризации**, который содержит пустые поля количеств для ввода подсчитанных запасов. После того как вы завершили подсчет и ввели количества на странице **Запись инвентаризации**, выберите действие **Готово**. Это действие переносит количества в соответствующие строки на странице **Заказ инвентаризации**. [!INCLUDE [prod_short](includes/prod_short.md)] обеспечивает, чтобы результаты подсчета товаров нельзя было записать дважды.  

> [!NOTE]
> Использование документов для выполнения инвентаризации обеспечивает дополнительный контроль и поддерживает распределение инвентаризации между несколькими сотрудниками. Для выполнения задачи также можно использовать журналы, например страницы **Журналы инвентаризации** и **Журнал инвентаризации склада**. Подробнее см. в статье [Подсчет, корректировка и повторная классификация запасов с помощью журналов](inventory-how-count-adjust-reclassify.md). В этой статье описывается, как выполнять инвентаризацию с использованием документов.
>
> Если на вашем складе используется зоны, использовать заказы инвентаризации невозможно. Вместо этого используйте страницу **Журнал инвентаризации склада** для инвентаризации складских записей перед их синхронизацией с операциями книги товаров.

Инвентаризация с использованием документов состоит из следующих этапов:

1. Создайте заказ инвентаризации с уже заполненными ожидаемыми количествами товаров.
2. Создайте одну или несколько записей инвентаризации из заказа.
3. Введите подсчитанные значения количества товаров в записи, как указано, например, в распечатках, и задайте значение **Завершен**.
4. Завершите и выполните учет заказа инвентаризации.

## <a name="to-create-a-physical-inventory-order"></a>Создание заказа инвентаризации

Заказ инвентаризации представляет собой готовый документ, который состоит из заголовка заказа инвентаризации и строк заказа. Информация в заголовке инвентаризации описывает, как выполнять инвентаризацию. Строки заказа содержат информацию о товарах и их местонахождении.

Для создания строк заказа инвентаризации обычно используется действие **Рассчитать строки**, которое добавляет текущие запасы в виде строк в заказ. Также можно использовать действие **Копировать из документа**, чтобы заполнить строки содержимым другого открытого или учтенного заказа инвентаризации. В следующей процедуре описывается только использование действия **Рассчитать строки**.

1. Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **Заказы инвентаризации**, а затем выберите связанную ссылку.
2. Выберите действие **Создать**.
3. Заполните требуемые поля на экспресс-вкладке **Общее**. [!INCLUDE[tooltip-inline-tip](includes/tooltip-inline-tip_md.md)]
4. На вкладке **Действия**  в группе **Функции**  выберите **Другое**, а затем выберите **Вычислить линии**.
5. Выберите требуемые параметры.
6. Установите фильтры, например чтобы включить только подмножество товаров, подлежащих инвентаризации с первой записью.

    > [!TIP]
    > Чтобы запланировать инвентаризацию запасов силами нескольких сотрудников, устанавливайте разные фильтры при каждом использовании действия **Рассчитать строки**, чтобы заполнять заказ только подмножеством товаров, который будет записывать данный пользователь. В этом случае при создании нескольких записей инвентаризации для нескольких сотрудников сводится к минимуму риск двойного учета товаров. Подробнее см. в разделе [Создание записи инвентаризации](#to-create-a-physical-inventory-recording).

7. Нажмите кнопку **ОК**.

Строка для каждого товара, который хранится на выбранном складе и соответствует заданным фильтрами и параметрам, добавляется в заказ. Для товаров, которые настроены для трассировки товаров, устанавливается флажок **Использовать трассировку товара**, и сведения об ожидаемом количестве по серийным номерам и номерам партий доступны при выборе действия **Строки** и затем **Строки трассировки товаров**. Подробнее см. в разделе [Обработка трассировки товаров при инвентаризации запасов](#handle-item-tracking-when-counting-inventory).

Теперь можно приступать к созданию одной или нескольких записей, которые являются инструкциям для сотрудников, который выполняют подсчет.  

> [!NOTE]
> Если вы используете трассировку конкретных пакетов, вы также можете использовать документы для подсчета и корректировки запасов с номерами пакетов. Чтобы начать использовать эту функцию, необходимо включить **Обновление функции: включение использования трассировки пакетов в заказах инвентаризации** на странице **Управлении функциями**. Существующие заказы инвентаризации будут обновлены, однако [!INCLUDE [prod_short](includes/prod_short.md)] не может заполнять поле **№ пакета** . Эти строки необходимо создать заново с помощью действия **Рассчитать строки** на странице **Заказ инвентаризации**.
>
> Ввести номера пакетов для товаров, которые требуют трассировки пакетов, можно на странице **Строки записи инвентаризации**. Выберите **Готово**, чтобы завершить запись.
>
> После того, как вы выберете **Готово** на странице **Заказ инвентаризации**, [!INCLUDE [prod_short](includes/prod_short.md)] вычислит разницы с учетом номеров пакетов и других данных для трассировки товаров, и внесет положительные или отрицательные корректировки.

## <a name="to-create-a-physical-inventory-recording"></a>Создание записи инвентаризации

Для каждого заказа инвентаризации можно создать один или несколько документов записи инвентаризации, в которые сотрудники вносят подсчитанные количества. Сотрудники могут вводить количества вручную или с помощью сканирующего устройства.

По умолчанию создается запись для всех строк соответствующего заказа инвентаризации. Во избежание подсчета двумя разными сотрудниками одних и тех товаров при распределении инвентаризации заполняйте заказ инвентаризации, устанавливая фильтры в пакетном задании **Рассчитать строки**. Затем создайте запись инвентаризации, установив флажок **Только строки, отсутствующие в записи**. Эти параметры обеспечивают, что каждая новая создаваемая запись будет содержать только товары, отличные от товаров в других записях.

В случае инвентаризации вручную можно напечатать отчет **Запись инвентаризации**, в котором имеется пустой столбец, куда сотрудники склада могут записывать подсчитанные количества. После завершения инвентаризации вы вводите записанные количества в связанные строки на странице **Запись инвентаризации**. Наконец, вы переносите зарегистрированные количества в соответствующих заказ инвентаризации, задав статус **Завершен**.

1. На странице **Заказа на инвентаризацию**, содержащей строки для товаров, которые необходимо подсчитать в одной записи, вам необходимо выбрать действие **Создать новую запись** .
1. На вкладке **Действия**  в группе **Функции**  выберите **Другое**, а затем выберите **Создать новую запись**.
1. Выберите требуемые параметры и задайте фильтры.
1. Нажмите кнопку **ОК**.
1. Для каждого набора товаров для подсчета загрузите их в соответствующий заказ инвентаризации и повторите шаги с 1 по 3 с установленным флажком **Только строки, отсутствующие в записи**.
1. На вкладке **Связанные** выберите **Заказ**, а затем выберите действие **Записи**, чтобы открыть страницу **Список записей инвентаризации физических веществ** .
1. Откройте соответствующую запись.
1. На экспресс-вкладке **Общее** заполните необходимые поля.
1. Для товаров, использующих трассировку товаров, создайте дополнительную строку для каждого номера партии или кода серийных номеров, выбрав действие **Функции**, затем действие **Копировать строку**. Подробнее см. в разделе [Обработка трассировки товаров при инвентаризации запасов](#handle-item-tracking-when-counting-inventory).  
1. Выберите действие **Печать**, чтобы подготовить физический документ, который сотрудники будут использовать для записи подсчитанных количеств.

## <a name="to-finish-a-physical-inventory-recording"></a>Завершение записи инвентаризации

После того как сотрудники подсчитают количества, запишите их в [!INCLUDE [prod_short](includes/prod_short.md)].

1. Со страницы **Список записей инвентаризации** выберите запись инвентаризации, которую требуется завершить, затем выберите действие **Правка**.
2. На экспресс-вкладке **Строки** заполните фактическое подсчитанное количество в поле **Количество** для каждой строки.
3. Для товаров с серийными номерам или номерами партий (установлен флажок **Использовать трассировку товаров**) введите подсчитанные количества в строках для серийных номеров и номеров партий соответственно. Подробнее см. в разделе [Обработка трассировки товаров при инвентаризации запасов](#handle-item-tracking-when-counting-inventory).
4. Установите флажок **Записано** в каждой строке.
5. После ввода всех данных для записи инвентаризации выберите действие **Готово**. Обратите внимание, что во всех строках должен быть установлен флажок **Записано**.

    > [!NOTE]
    > После завершения записи инвентаризации каждая строка перемещается в строку соответствующего заказа инвентаризации, которая ей точно соответствует. Для соответствия значения в полях **Код товара**, **Код варианта**, **Код склада** и **Код ячейки** должны быть одинаковыми в строках записи и заказа.>
    > 
    > Если соответствующая строка заказа инвентаризации не существует и если флажок **Разрешить запись без заказа** установлен, автоматически добавляется новая строка, а в соответствующей строке заказа инвентаризации устанавливается флажок **Записано без заказа**. В противном случае выводится сообщение об ошибке, и процесс отменяется.> Если несколько строк записи инвентаризации соответствуют строке заказа инвентаризации, отображается сообщение и процесс отменяется. Если по какой-либо причине две одинаковые строки инвентаризации попадают в заказ инвентаризации, существует действие, которое можно использовать для разрешения ситуации. Подробнее см. в разделе [Поиск дублирующихся строк заказа инвентаризации](#to-find-duplicate-physical-inventory-order-lines).

## <a name="to-complete-a-physical-inventory-order"></a>Завершение заказа инвентаризации

После завершения регистрации физической инвентаризации поле **Количество. Регистратор (база)**  в соответствующем заказе на физическую инвентаризацию обновляется подсчитанными (записанными) значениями, а флажок **На строках регистрации**  устанавливается. Если подсчитанное количество отличается от ожидаемого, то в полях **Полож. кол-во (база)** и **Отрицат. кол-во (база)** отображается разница.

Для просмотра ожидаемых количеств и записанных разниц для товаров с трассировкой выберите действие **Строки**, а затем выберите действие **Строки трассировки товаров** для выбора различных представлений для серийных номеров и номеров партий в инвентаризации.

Также можно выбрать действие **Разница заказов инвентаризации** действие для просмотра возможных разниц между ожидаемым количеством и подсчитанным количеством.

### <a name="to-find-duplicate-physical-inventory-order-lines"></a>Поиск дублирующихся строк заказа инвентаризации

1. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **Заказы инвентаризации**, а затем выберите связанную ссылку.
2. Откройте заказ инвентаризации, для которого вы хотите просмотреть дублирующиеся строки.
1. На вкладке **Связанные**  выберите **Другие**, а затем выберите **Показать повторяющиеся строки**.

Отображаются повторяющиеся строки заказа инвентаризации, чтобы вы могли их удалить и оставить только одну строку с уникальным набором значений в полях **Номенклатурный номер**, **Код варианта**, **Код склада** и **Код ячейки**.

### <a name="to-post-a-physical-inventory-order"></a>Учет заказа инвентаризации

После выполнения заказа инвентаризации и изменения его статуса на **Завершено** можно учесть его. Установить для заказа инвентаризации статус **Завершено** можно только при выполнении следующих условий:

- Все связанные записи инвентаризации имеют статус **Завершен**.
- Каждая строка заказа инвентаризации подсчитана по крайней мере в одной строке записи инвентаризации.
- Для всех строк заказа инвентаризации установлены флажки **В строках записей** и **Расчетное ожидаемое кол-во**.

1. Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **Заказы инвентаризации**, а затем выберите связанную ссылку.
2. Выберите заказ инвентаризации, который требуется завершить, затем выберите действие **Изменить**.

    На странице **Заказ инвентаризации** записанное количество отображается в поле **Записанное кол-во (баз.)**.
3. Выберите действие **Готово**.

    Значение в поле **Статус** меняется на **Завершено**. Теперь внести изменения в заказ можно, только выбрав действие **Открыть повторно**.
4. Для учета заказа выберите действие **Учет**, затем выберите кнопку **ОК**.

    Записи книги товаров будут обновлены вместе со всеми связанными записями трассировки товаров.

    [!INCLUDE [preview-posting-inventory](includes/preview-posting-inventory.md)]

### <a name="to-view-posted-physical-inventory-orders"></a>Просмотр учтенных заказов инвентаризации

После учета заказ инвентаризации будет удален, и вы сможете просматривать и оценивать документ как учтенный заказ инвентаризации. Учтенный заказ включает в себя записи инвентаризации и комментарии, если они были добавлены.

1. Выберите значок ![Лампочка, которая открывает функцию «Что вы хотите сделать»](media/ui-search/search_small.png "Что вы хотите сделать"), значок, введите **Учтенные заказы инвентаризации**, а затем выберите связанную ссылку.
2. На странице **Учтенные заказы инвентаризации** выберите учтенный заказ инвентаризации, который требуется просмотреть, затем выберите действие **Просмотреть**.
3. На вкладке **Связанные**  выберите **Заказ**, а затем выберите действие **Записи**, чтобы просмотреть список связанных записей физической инвентаризации.  

## <a name="handle-item-tracking-when-counting-inventory"></a>Обработка трассировки товаров при инвентаризации запасов

Трассировка товаров относится к серийным номерам или номера партий, назначенным товарам. При инвентаризации товара, который хранится в запасах как, например, 10 различных номеров партий, сотрудник должен быть в состоянии записать, какие и сколько единиц каждого номера партии находятся на складе. Подробнее см. в разделе [Работа с серийными номерами и номерами партий](inventory-how-work-item-tracking.md).

Флажок **Использовать трассировку товаров** в строках заказа инвентаризации устанавливается автоматически, если для товара настроен код трассировки. Вы можете установить или снять этот флажок вручную.

### <a name="example---prepare-a-physical-inventory-recording-for-an-item-tracked-item"></a>Пример. Подготовка записи инвентаризации для товара с трассировкой

Рассмотрим инвентаризацию для товара А, который хранится в запасах как десять различных серийных номеров.

1. В строке записи для товара установите флажок **Использовать трассировку товаров**.
2. Выберите поле **Серийный номер**, выберите первый серийный номер, который существует в запасах для товара, затем нажмите кнопку **ОК**.

    Скопируйте строку для первого товара с трассировкой, чтобы вставить дополнительные строки, соответствующие количеству серийных номеров, которые хранятся на складе.

3. Выберите действие **Функции**, затем действие **Копировать строку**.
4. На странице **Копировать строку записи инвентаризации** введите **9** в поле **Количество копий**, затем нажмите кнопку **ОК**.
5. В первой строке выберите поле **Серийный №** и выберите следующий серийный номер для товара.
6. Повторите шаг 5 для остальных восьми серийных номеров. Следите за тем, чтобы не выбрать один и тот же номер дважды.
7. Выберите действие **Печать**, чтобы подготовить распечатку, которую сотрудники будут использовать для записи подсчитанных товаров и серийных номеров или номеров партий.

Обратите внимание, что отчет **Запись инвентаризации** содержит десять строк для товара А, по одной для каждого серийного номера.

### <a name="example---record-and-post-counted-lot-number-differences"></a>Пример. Запись и учет разницы инвентаризированных номеров партий

Товар с отслеживанием партии-хранится на складе с серией номеров "LOT".

**Ожидаемые запасы**:

|Номер партии|количество;|
|-|-|
|LOT1001|80|
|LOT1003|30|
|LOT1006|10|
|Итого|120|

**Зарегистрированные количества**:

|Номер партии|количество;|
|-|-|
|LOT1001|80|
|LOT0002|12|
|LOT1003|20|
|LOT1006|10|
|Итого|112|

**Количества к учету**:

|Номер партии|Ожидаемое количество|Зарегистрированное количество|Количество к учету|
|-|-|-|-|
|LOT1001|80|80|0|
|LOT1002|0|12|+12|
|LOT1003|30|20|-10|
|LOT1006|10|0|-10|
|Итого|120|112|–8|

На странице **Заказ инвентаризации** поле **Отриц. кол-во (баз.)** содержит **8**. Для данной строки заказа на странице **Список трассировки товара для инвентаризации** отображаются положительные или отрицательные количества для каждого номера партии.

## <a name="inventory-documents"></a>Товарные документы

Следующие типы документов полезны при управлении складом:

* Используйте **Приходные накладные** для регистрации положительных корректировок товаров в зависимости от качества, количества и стоимости.
* Используйте **Расходные накладные** для списания недостающих или поврежденных товаров.

Вы можете распечатать эти документы на любом этапе, выпустить и снова открыть их, а также назначить общие значения, такие как размеры, в заголовке. Чтобы перепечатать документы после их учета, используйте страницы **Учтенная приходная накладная** и **Учтенная расходная накладная**.

> [!NOTE]
> Прежде чем вы сможете использовать эти документы, вы должны указать номерную серию для создания их идентификаторов. Подробнее см. в разделе [Настройка нумерации товарных документов](#to-set-up-numbering-for-inventory-documents).

### <a name="to-set-up-numbering-for-inventory-documents"></a>Настройка нумерации товарных документов

Ниже приводится процедура настройки нумерации для товарных документов.

1. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **Настройка запасов**, а затем выберите связанную ссылку.
2. На экспресс-вкладке **Нумерация** укажите в следующих полях серии номеров документов.

   - **Номера квитанций инв.**  
   - **Размещенные инв. квитанции №№**  
   - **Инв. номера отгрузок**  
   - **Размещенные инв. номера отгрузок**  

### <a name="to-create-and-post-an-inventory-document"></a>Чтобы создать и учесть инвентарный документ

В следующей процедуре показан порядок создания, печати и учета приходной накладной. Действия для расходных накладных аналогичны.

1. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок введите **Приходные накладные**, а затем выберите связанную ссылку.  
2. В заголовке страницы **Приходная накладная**  выберите местоположение в поле **Код местоположения**, а затем заполните оставшиеся поля по мере необходимости.
3. На экспресс-вкладке **Строки** в поле **Товар** выберите товар в запасах. В поле **Кол-во** укажите количество товаров для добавления.
4. Чтобы распечатать отчет **Приходная накладная** со страницы **Приходная накладная**, выберите действие **Печать** .

На странице **Приходная накладная**  доступны следующие функции:

- Выберите действие **Выпустить** или **Открыть повторно** для задания статуса для следующего этапа обработки.  
- Выберите действие **Учесть**, чтобы учесть приходную накладную, или выберите **Учесть и распечатать**, чтобы учесть приход и распечатать тестовый отчет.

## <a name="printing-inventory-documents"></a>Печать товарных документов

Вы можете указать отчеты, которые необходимо печатать на разных этапах, выбрав один из следующих вариантов в поле **Использование** страницы **Выбор отчета — запасы**:

* Приходная накладная
* Расходная накладная
* Учтенная приходная накладная
* Учтенная расходная накладная

> [!NOTE]
> Набор доступных отчетов может быть разным в зависимости от локализации для вашей страны/региона. Базовое приложение не содержит никаких макетов.
