## <a name="to-view-archived-workflow-step-instances"></a><span data-ttu-id="1840c-109">Просмотр архивированных экземпляров шагов потоков операций</span><span class="sxs-lookup"><span data-stu-id="1840c-109">To view archived workflow step instances</span></span>  
1.  <span data-ttu-id="1840c-110">Выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Рабочие процессы**, затем выберите связанную ссылку.</span><span class="sxs-lookup"><span data-stu-id="1840c-110">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Workflows**, and then choose the related link.</span></span>  
2.  <span data-ttu-id="1840c-111">Откройте поток операций, для которого нужно просмотреть все помещенные в архив экземпляры шагов потоков операций.</span><span class="sxs-lookup"><span data-stu-id="1840c-111">Open the workflow for which you want to view all archived workflow step instances.</span></span>  
3.  <span data-ttu-id="1840c-112">В окне **Рабочий процесс** выберите действие **Архивные экземпляры шагов рабочих процессов**.</span><span class="sxs-lookup"><span data-stu-id="1840c-112">In the **Workflow** window, choose the **Archived Workflow Step Instances** action.</span></span>  

    <span data-ttu-id="1840c-113">Откроется окно **Архивные экземпляры шагов рабочих процессов**, содержащее архивные экземпляры шагов выбранного рабочего процесса.</span><span class="sxs-lookup"><span data-stu-id="1840c-113">The **Archived Workflow Step Instances** window opens showing the archived workflow step instances of the selected workflow.</span></span>  
4.  <span data-ttu-id="1840c-114">Можно также выбрать значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), ввести **Архивные экземпляры шагов рабочих процессов**, затем выбрать связанную ссылку.</span><span class="sxs-lookup"><span data-stu-id="1840c-114">Alternatively, Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Archived Workflow Step Instances**, and then choose the related link.</span></span>  

<span data-ttu-id="1840c-115">Откроется окно **Архивные экземпляры шагов рабочих процессов**, содержащее все архивные экземпляры шагов рабочих процессов.</span><span class="sxs-lookup"><span data-stu-id="1840c-115">The **Archived Workflow Step Instances** window opens showing all archived workflow step instances.</span></span> <span data-ttu-id="1840c-116">В поле **Код рабочего процесса** можно посмотреть, к какому потоку операций относится архивированный шаг рабочего процесса.</span><span class="sxs-lookup"><span data-stu-id="1840c-116">In the **Workflow Code** field, you can see which workflow the archived workflow step instance belonged to.</span></span>
