# <a name="insure-fixed-assets"></a>Страхование основных средств

Используйте страницу  **Карта страхования**, чтобы настроить страховой полис для покрытия одного или нескольких основных средств. Вы можете назначить одно основное средство одному страховому полису или несколько основных средств одному страховому полису.

Можно назначить основное средство страховому полису путем учета в книге страхового покрытия со страницы **Журнал страхования**.

Кроме этого, можно назначить основное средство страховому полису и создать записи книги страхования при учете стоимости приобретения. Вы проводите стоимость приобретения из журнала основных средств с заполненным полем  **Номер страхования** . Необходимо включить переключатель **Автоматическая проводка по страхованию** на странице **Настройка основных средств** . Для получения дополнительной информации см.  [Приобретение основного средства с помощью журнала главной книги основных средств](fa-how-acquire.md#acquire-a-fixed-asset-by-using-a-fixed-asset-gl-journal).

Если переключатель **Автоматическая проводка по страхованию** на странице **Настройка основных средств** не включен, учет приобретений из Журнал основных средств создает строки на странице  **Журнал страхования** . Вы должны опубликовать эти строки вручную.

> [!WARNING]  
> Если вы не включите  **Автоматическую проводку по страхованию** переключитесь на страницу  **Настройка основных средств**, ваш страховой журнал должна быть основана на шаблоне журнала без номерной серии. Это связано с тем, что в противном случае вставленные номера документа из строки журнала основного средства будут противоречить серии номеров журнала страхования. Для получения дополнительных сведений о шаблонах журналов и разделах см. раздел [Настройка общей информации об основных средствах](fa-how-setup-general.md).

После назначения основного средства страховому полису поле  **Застрахован**  на карточке основного средства содержит  **Да**. Когда вы продаете основное средство, переключатель автоматически отключается.

## <a name="to-create-or-modify-an-insurance-card"></a>Создание или изменение страховой карточки

При получении информации об изменениях в сумме покрытия необходимо ввести новую информацию на странице **Страховая карточка** для проверки правильности анализа покрытия страхового полиса.  

1. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **Страхование**, а затем выберите связанную ссылку.
2. Выберите действие **Создать** для создания новой карты для страхового полиса. [!INCLUDE[tooltip-inline-tip](includes/tooltip-inline-tip_md.md)]
3. Можно также выбрать политику страхования, которую необходимо изменить, затем выбрать действие **Правка**.

## <a name="to-assign-a-fixed-asset-to-an-insurance-policy-by-posting-from-the-insurance-journal"></a>Назначение основного средства страховому полису путем разноски из журнала страхования

Основное средство назначается страховому полису путем разноски в книгу страхового покрытия.  

В следующей процедуре объясняется, как вручную создать строку журнала страхования. Если переключатель  **Автоматическая проводка по страхованию**  включен на странице  **Настройка основных средств**, строки журнала страхования создаются автоматически, когда вы учитываете затраты на приобретение. В таком случае все, что необходимо сделать, это выполнить учет журнала.  

1. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **Журналы страхования**, а затем выберите связанную ссылку.  
2. Откройте соответствующий журнал и заполните строки журнала, как требуется.  
3. Чтобы назначить несколько основных средств одному страховому полису, создайте строки журнала с тем же значением в поле **Номер страхования** и другими значениями в поле **Номер ОС**.  
4. Выберите действие **Учет**.  

    > [!NOTE]  
    > Операции из журнала страхования учитываются только в книгу страхования.  

## <a name="to-update-the-insurance-value-of-a-fixed-asset"></a>Для обновления суммы страхования основного средства

Можно использовать пакетное задание **Индекс страхования** для обновления покрываемых страховкой основных средств.  

1. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **Индекс страхования**, а затем выберите связанную ссылку.
2. Заполните соответствующим образом поля.

    > [!NOTE]  
    >   В поле **Цифра индекса** значение уменьшения на, например, 5% вводится как 95, а значение увеличения на 2% вводится как 102.  
3. Нажмите кнопку **ОК**.  

   Пакетное задание рассчитывает новую сумму как процент от общей страховой суммы, указанной на странице **Статистика страхования**, и создает строку в страховом журнале.  
4. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **Журналы страхования**, а затем выберите связанную ссылку.  
5. Откройте соответствующий журнал страхования, просмотрите созданные значения, затем учтите их в книге страхового покрытия.  

## <a name="to-monitor-insurance-coverage"></a>Мониторинг страхового покрытия

В [!INCLUDE[prod_short](includes/prod_short.md)] предусмотрены специальные страницы отчетов и статистики для использования при анализе страховых полисов и сумм страхового покрытия для основных средств (не являются ли они недостаточными или избыточными).  

### <a name="overview-of-insurance-policies"></a>Обзор страховых полисов

Чтобы просмотреть сведения о страховых полисах, предварительно просмотрите или напечатайте отчет **Страхов. - список**. Отчет показывает все полисы и наиболее важные поля из страховых карточек.  

### <a name="insurance-coverage"></a>Страховое покрытие

Чтобы посмотреть, какие страховые полисы покрывают средство и в какой сумме, можно просмотреть или напечатать отчет **Страхов. - общая застрах. стоим.**.  

#### <a name="overunder-coverage"></a>Больше/недостаточное покрытие

Проверить, застрахованы ли основные средства чрезмерно или недостаточно, можно следующими способами:  

* Страница **Статистика страхования**. Положительная сумма в поле **Недострахованный/перестрахованный** означает, что основное средство перестраховано. Отрицательная сумма означает, что актив недостаточно застрахован.  
* Страница **Статистика ОС**. Выберите поле **Общая страховая сумма** для просмотра страницы **Книга операций по страхованию**.  
* Отчет **Избыточное/недостаточное покрытие**.  
* Отчет **Страхование - анализ**.  

### <a name="uninsured-fixed-assets"></a>Незастрахованные основные средства

Чтобы проверить, не забыли ли вы присвоить основное средство страховому полису, вы можете распечатать или просмотреть отчет  **Страхование - Незастрахованные ОС** . В этом отчете отображаются основные средства, суммы по которым не учитываются в книге страхового покрытия.  

## <a name="to-view-insurance-coverage-ledger-entries"></a>Просмотр операций книги страхования

Вы можете просмотреть записи, которые вы сделали в книге страхового покрытия.  

1. Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **Страхование**, а затем выберите связанную ссылку.  
2. Выберите нужный полис страхования и выберите действие **Книга операций по страхованию**.  

## <a name="to-view-the-total-insurance-value-of-fixed-assets"></a>Просмотр общих страховых значений для основных средств

На странице матрицы показаны страховые стоимости, зарегистрированные для каждого страхового полиса для каждого основного средства, полученные в результате учтенных сумм, связанных со страхованием.  

1. Выберите значок ![Лампочка, которая открывает функцию «Что вы хотите сделать»](media/ui-search/search_small.png "Что вы хотите сделать"), значок, введите **Страхование**, а затем выберите связанную ссылку.  
2. Выберите соответствующий страховой полис, затем выберите действие **Общая страховая сумма по ОС**.  
3. Заполните соответствующим образом поля.  
4. Выберите действие **Показать матрицу**.  
5. Для просмотра операций книги страхования выберите значение в матрице.  

## <a name="to-correct-insurance-coverage-entries"></a>Исправление операций страхового покрытия

Если основное средство было присвоено неправильному страховому полису, это можно исправить, создав две записи реклассификации из журнала страхования.  

1. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **Журналы страхования**, а затем выберите связанную ссылку.  
2. Создайте одну строку журнала для основного средства и правильного страхового полиса, в которой поле **Сумма** содержит положительное значение.  
3. Создайте вторую строку журнала для основного средства и неправильного страхового полиса, в которой поле **Сумма** содержит отрицательное значение.  
4. Выберите действие **Учет**.  

Основное средство снимается с неправильного страхового полиса по второй строке. Актив присваивается правильному страховому полису в первой строке журнала.
