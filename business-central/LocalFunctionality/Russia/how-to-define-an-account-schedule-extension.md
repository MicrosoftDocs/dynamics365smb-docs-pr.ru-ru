## <a name="to-define-an-account-schedule-extension"></a><span data-ttu-id="01f4b-106">Определение расширения финансового отчета</span><span class="sxs-lookup"><span data-stu-id="01f4b-106">To define an account schedule extension</span></span>

1. <span data-ttu-id="01f4b-107">Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать](../../media/ui-search/search_small.png "Что вы хотите сделать"), введите **Финансовый отчет**, а затем выберите соответствующую ссылку.</span><span class="sxs-lookup"><span data-stu-id="01f4b-107">Choose the ![Lightbulb that opens the Tell Me feature](../../media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Account Schedule**, and then choose the related link.</span></span>
2. <span data-ttu-id="01f4b-108">Выберите финансовый отчет, для которого необходимо определить расширение.</span><span class="sxs-lookup"><span data-stu-id="01f4b-108">Select the account schedule for which you want to define an extension.</span></span>
3. <span data-ttu-id="01f4b-109">Выберите **Изменение финансового отчета**.</span><span class="sxs-lookup"><span data-stu-id="01f4b-109">Select **Edit account schedule**.</span></span>
4. <span data-ttu-id="01f4b-110">В поле **Номер строки**</span><span class="sxs-lookup"><span data-stu-id="01f4b-110">In the **Row No.**</span></span> <span data-ttu-id="01f4b-111">выберите номер строки финансового отчета, для которого необходимо определить расширение.</span><span class="sxs-lookup"><span data-stu-id="01f4b-111">field, select the number of the account schedule row for which you want to define an extension.</span></span>
5. <span data-ttu-id="01f4b-112">В поле **Тип группировки**, выберите **Пользовательский**, затем выберите таблицу, которая должна использоваться, из поля **Исходная таблица расширения**.</span><span class="sxs-lookup"><span data-stu-id="01f4b-112">In the **Totaling Type** field, select **Custom**, and then select the table that you want to use from the **Extension Source Table** field.</span></span>

<span data-ttu-id="01f4b-113">На основе вашего выбора сведения финансового отчета фильтруются из таблиц операций, а затем новые суммы рассчитываются для указанной строки финансового отчета.</span><span class="sxs-lookup"><span data-stu-id="01f4b-113">Based on your selections, the account schedule information is filtered from the entry tables, and then new amounts are calculated for the specified account schedule line.</span></span>
