# <a name="set-up-a-quantity-book"></a>Настройка книги количественного учета

Создайте книгу амортизации для хранения записей количеств основных средств, для которых во время ввода в эксплуатацию будет списываться сумма, по которой они приобретены.

## <a name="to-set-up-a-quantity-book"></a>Настройка книги количественного учета

1. Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать](../../media/ui-search/search_small.png "Что вы хотите сделать"), введите **Книги амортизации**, а затем выберите соответствующую ссылку.
2. В поле **Тип книги учета** выберите **Аналитическая**.
3. На экспресс-вкладке **Интеграция** проверьте, что никакие параметры не выбраны.
4. Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать](../../media/ui-search/search_small.png "Что вы хотите сделать"), введите **Настройка ОС**, а затем выберите соответствующую ссылку.
5. В поле **Книга аморт. - количеств.** введите код созданной книги амортизации.
6. Нажмите кнопку **ОК**.
