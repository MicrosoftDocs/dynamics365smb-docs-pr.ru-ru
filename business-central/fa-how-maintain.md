# <a name="maintain-fixed-assets"></a>Обслуживание основных средств

Затраты на обслуживание — это операционные расходы на мероприятия, направленные на поддержание состояния ваших основных средств. В отличие от капитального ремонта, техническое обслуживание не увеличивает стоимость основного средства.

Всякий раз, когда вы отправляете основное средство на обслуживание, вы фиксируете соответствующую информацию, такую как дата обслуживания, поставщика, который выполняет обслуживание, и телефонный номер сервисного агента. Информация о техническом обслуживании вводится на нескольких страницах:

* Карточка ОС
* Заказ на покупку
* Счет покупки
* ОС - журнал ГК

Переоценка используется для коррекции значений при изменении общего уровня цен. Для пересчета затрат на обслуживание используется пакетное задание **Индекс ОС**.

## <a name="record-a-maintenance-cost-directly-on-a-fixed-asset"></a>Запись затрат на обслуживание непосредственно в основном средстве

Каждый раз при проведении обслуживания (например, при сервисном посещении) основного средства вы можете записать его на странице **Регистрация обслуживания**.  

1. Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок введите **Основные Средства**, а затем выберите связанную ссылку.  
2. Выберите основное средство, для которого требуется зарегистрировать обслуживание, затем выберите действие **Обслуживание - регистрация**.
3. На странице **Обслуживание - регистрация** заполните требуемые поля. [!INCLUDE[tooltip-inline-tip](includes/tooltip-inline-tip_md.md)]  

## <a name="post-maintenance-costs-from-a-fixed-asset-gl-journal"></a>Учет затрат на обслуживание из журнала ГК учета основных средств

1. Выберите значок ![Лампочка, которая открывает функцию «Что вы хотите сделать»](media/ui-search/search_small.png "Что вы хотите сделать"), значок, введите **Список книг амортизации**, а затем выберите связанную ссылку.  
2. Выберите книгу амортизации, которая назначена основному средству, затем выберите действие **Правка**.
3. На странице **Карточка книги амортизации** убедитесь, что флажок **Обслуживание** не установлен, чтобы не отразить расходы на обслуживание в главной книге.
4. Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **Журналы ГК для ОС**, а затем выберите связанную ссылку.  
5. Создайте начальную строку журнала и заполните поля, как требуется.
6. В поле **Тип учета ОС** выберите **Обслуживание**.
7. Выберите действие **Вставить баланс. счет ОС**. Вторая строка журнала создается для балансирующего счета, который настроен для учета обслуживания.

    > [!NOTE]  
    > Шаг 7 работает только при условии, что на странице **Карточка учетной группы ОС** для учетной группы основного средства поле **Счет обслуживания** содержит дебетовый счет главной книги, а поле **Баланс. счет обслуживания** содержит счет главной книги, на котором требуется учитывать балансирующие операции для повышения стоимости. Дополнительные сведения см. в разделе [Настройка учетных групп основных средств](fa-how-setup-general.md#to-set-up-fixed-asset-posting-groups).
8. Выберите действие **Учет**.

## <a name="record-maintenance-cost-from-a-purchase-invoice"></a>Запись затрат на обслуживание из счета покупки

Следующие шаги описывают, как записать затраты на обслуживание основного средства из счета покупки. Действия для заказов на покупку аналогичны.

1. Выберите значок ![Лампочка, которая открывает функцию «Что вы хотите сделать»](media/ui-search/search_small.png "Что вы хотите сделать"), значок введите **Счет покупки**, а затем выберите связанную ссылку.
2. На экспресс-вкладке **Общее** заполните необходимые поля. [!INCLUDE[tooltip-inline-tip](includes/tooltip-inline-tip_md.md)]
3. В поле **Тип** на экспресс-вкладке **Строки** выберите **Основное средство**.
4. В поле **Номер** выберите основное средство, а затем укажите количество и стоимость.
5. В поле **Тип учета ОС** выберите **Обслуживание**.
6. Выполните операцию учета счета покупки.

## <a name="follow-up-on-service-visits"></a>Контроль сервисных посещений

Для получения списка активов, для которых запланировано сервисное посещение, можно напечатать отчет **Обслуживание - след. сервис**. Этот отчет также можно использовать для обновления поля **След. дата сервисного обслуживания** на карточках основных средств.  

1. Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **Обслуживание - след. сервис**, а затем выберите связанную ссылку.  
2. Заполните поля **Дата начала** и **Дата окончания**.  
3. Нажмите кнопку **Печать** или **Просмотр**.

## <a name="monitor-maintenance-costs"></a>Мониторинг затрат на обслуживание

Для мониторинга затрат на обслуживание можно просматривать статистику.  

1. Выберите значок ![Лампочка, которая открывает функцию «Что вы хотите сделать»](media/ui-search/search_small.png "Что вы хотите сделать"), значок введите **Основные Средства**, а затем выберите связанную ссылку.
2. Выберите основное средство, для которого требуется просмотреть затраты на обслуживание, затем выберите действие **Книги амортизации**.
3. На странице **Книги амортизации ОС** выберите книгу амортизации соответствующего основного средства, затем выберите действие **Статистика**.
4. На странице **ОС - статистика** выберите поле **Обслуживание**.

Страница **Операции книги обслуживания** позволяет просмотреть операции, из которых складывается сумма в поле **Обслуживание**.

## <a name="view-or-print-maintenance-costs-for-multiple-fixed-assets"></a>Просмотр или печать затрат на обслуживание для нескольких основных средств

Отчет **Обслуживание - анализ** можно использовать для просмотра обслуживания по одному, двум или трем кодам обслуживания на определенную дату или за определенный период. Можно посмотреть итоги по всем выбранным основным средствам или итоги по каждому основному средству.

1. Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **Анализ обслуживания**, а затем выберите связанную ссылку.
2. Заполните соответствующим образом поля.
3. Нажмите кнопку **Печать** или **Просмотр**.

## <a name="view-maintenance-ledger-entries"></a>Просмотр операций книги обслуживания

Можно также изучать затраты на обслуживание, просматривая операции книги обслуживания.  

1. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок введите **Основные Средства**, а затем выберите связанную ссылку.
2. Выберите основное средство, для которого требуется просмотреть операции книги учета, затем выберите действие **Книги амортизации**.
3. На странице **Книги амортизации ОС** выберите книгу амортизации соответствующего основного средства, затем выберите действие **Книга операций по обслуживанию**.

## <a name="view-or-print-maintenance-ledger-entries-for-multiple-fixed-assets"></a>Просмотр или печать операций книги обслуживания для нескольких основных средств

В отчете **Обслуживание - подробности** можно просмотреть или распечатать операции книги обслуживания для одного или нескольких основных средств.  

1. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **Обслуживание - подробности**, а затем выберите связанную ссылку.
2. Заполните соответствующим образом поля.
3. Нажмите кнопку **Печать** или **Просмотр**.
