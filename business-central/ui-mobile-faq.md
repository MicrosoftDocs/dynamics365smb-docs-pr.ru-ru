# <a name="mobile-apps-faq"></a>Вопросы и ответы по мобильным приложениям

В этой статье содержатся ответы на вопросы, которые часто задают наши опытные пользователи о мобильных приложениях для [!INCLUDE [prod_short](includes/prod_short.md)].  

## <a name="is-there-an-app-for-my-device"></a>Есть ли приложение для моего устройства?

Вероятно! Установите приложение [!INCLUDE[prod_short](includes/prod_short.md)] на мобильное устройство, скачав его из магазина Windows, App Store или Google Play.

- [Магазин Windows](https://go.microsoft.com/fwlink/?LinkId=734848) (только ПК)
- [App Store](https://go.microsoft.com/fwlink/?LinkId=734847)
- [Google Play](https://go.microsoft.com/fwlink/?LinkId=734849)

## <a name="is-it-the-same-experience-in-the-apps-as-in-the-browser"></a>В приложениях такой же опыт, как и в браузере?

Нет, не совсем так. Например, мы показываем только группу действий **Главная** из-за ограниченного размера экрана на мобильных устройствах. Аналогично, сочетания клавиш недоступны, потому что вы в основном используете сенсорный ввод, а не клавиатуру, для навигации на мобильных устройствах.

В следующей таблице описаны некоторые из наиболее распространенных различий и ограничений, с которыми вы можете столкнуться при использовании [!INCLUDE [prod_short](includes/prod_short.md)] на мобильных устройствах по сравнению с браузером.

| Основное понятие | На планшетах | На телефонах | Пример из браузера |
|--|--|--|--|
| Группы действий | Отображается только группа действий **Главная**. | Отображается только группа действий **Главная**. | **Главная** и **Размещенные документы** в ролевом центре `Sales Order Processor`. |  |
| Выбор нескольких записей в списках | Недоступно. | Недоступно. | `Ctrl+A` или `Ctrl+Click` в строках списка в браузере. |
| Действия на панели действий | Отображаются только повышенные действия. | Отображаются только повышенные действия. |  |
| Информационные панели | Не отображаются на страницах списка или страницах листов. | Не отображаются на страницах списка или страницах листов. | Список `Customer` в ролевом центре `Small Business`. |
| Расширенные фильтры | Фильтрация по столбцам недоступна. | Фильтрация по столбцам недоступна. | На странице списка `Customer`. |
| Что вы хотите сделать | Пока недоступно. | Пока недоступно. | См. раздел [Поиск страниц и информации с помощью функции "Что вы хотите сделать"](ui-search.md). |  |
| Обозреватель ролей | Пока недоступно. | Пока недоступно. | См. раздел [Поиск страниц с помощью обозревателя ролей](ui-role-explorer.md). |
| Поля на экспресс-вкладках | Поля на экспресс-вкладках страниц списков не отображаются. В области содержимого страницы отображается только элемент управления повторителем. | Недоступно. |  |
| Выбор из полного списка | Недоступно в подстановках. Пользователи не могут выполнять действия на странице подстановки и не могут получить доступ к полному набору записей. | Недоступно в подстановках. Пользователи не могут выполнять действия на странице подстановки и не могут получить доступ к полному набору записей. | В `Item Card` при выборе пункта **Базовые единицы измерения**. |
| Поиск по столбцам списка | Частично поддерживается. Поиск не будет включать поля FlowField. | Частично поддерживается. Поиск не будет включать поля FlowField. | См. примеры на странице списка `Customers`. |
| Подстановки | Доступно. | Доступно, с той разницей, что расширенная и простая подстановка на телефоне ведут себя одинаково. При подстановке не будет отображаться карточка, отображаться информационные панели или какие-либо группы полей. | См. примеры на странице `Customer Card`. |
| Матричные элементы управления | Недоступно. | Недоступно. | См. пример в `G/L Budget`. |
| Загрузка файлов | Доступно. Невозможно загрузить несколько файлов одновременно. | Доступно. Невозможно загрузить несколько файлов одновременно. | Отчет `Trial Balance` во флажке **Печатать в Excel**. |
| Страницы рабочих листов | Доступно. | Недоступно; отображается сообщение об ошибке. | Лист `Sales Price` или лист `Cash Flow`. |
| Списки | Доступно. | Доступны, с той разницей, что они отображаются в кирпичном макете. | Страницы клиентов или заказов на продажу. |
| Отступы в элементах управления повторителем | Доступно. | Недоступно. Элемент управления повторителем будет отображаться как обычный плоский кирпичный макет. | Страницы плана счетов и списка контактов. |
| Автоматический фокус ввода на первом редактируемом поле страницы | Недоступно. | Недоступно. | Страница `Customer Card`.<BR /><BR />В браузере фокус автоматически будет на первом редактируемом поле (например, поле `Name`), позволяя сразу же изменить значение.<BR /><BR />В приложениях для планшетов и телефонов это поле не будет в фокусе; вместо этого вам придется сначала вручную выбрать поле, чтобы внести изменения.|
