# Управление основными средствами

Функциональные возможности по управлению основными средствами [!INCLUDE[prod_short](includes/prod_short.md)] позволяют получать информацию об основных средствах и правильно начислять их амортизацию. Кроме того, они позволяют отслеживать затраты на обслуживание, управлять страховыми полисами, учитывать транзакции с основными средствами и формировать различные отчеты и статистику.

## Первичная настройка основных средств

Прежде чем вы сможете управлять основными средствами, необходимо выполнить следующие настройки:

- Значения по умолчанию
- Учет основных средств
- Учетные группы и типы учета
- Ключи распределения
- журналы основных средств;

Подробнее см. в статье [Настройка основных средств](fa-setup.md).

## Регистрация основных средств

Для каждого из основных средств необходимо настроить карточку, содержащую сведения об этом основном средстве. Например, можно настроить здания или производственное оборудование как комплекты ОС со списком компонентов. Основные средства можно группировать разными способами, например по классам, подразделениям или местоположению. После этого можно приобретать, обслуживать и продавать основные средства. Можно также настроить бюджетные активы. Бюджетирование позволит включать в отчеты любые ожидаемые приобретения и продажи.

| Создать основные средства, назначить методы амортизации, учесть приобретения, ликвидационную стоимость и напечатать списки основных средств. |[Приобретение основных средств](fa-how-acquire.md) |

## Настройка амортизации основных средств

Для отслеживания амортизации основных средств, а также других финансовых транзакций для основных средств, требуется настроить одну или несколько книг амортизации для каждого основного средства. Для амортизации основных средств необходимо выполнить несколько действий:

1. Запустить отчет для расчета периодической амортизации.
1. Заполните журнал получившимися операциями.
1. Выполните учет журнала.

[!INCLUDE[prod_short](includes/prod_short.md)] поддерживает несколько методов амортизации. Подробнее см. в статье [Методы амортизации](fa-depreciation-methods.md). Можно настроить несколько книг амортизации для каждого основного средства для различных целей, например, одну для налоговой отчетности, другую для внутренней отчетности.

## Обслуживание и страхование основных средств

Для каждого основного средства можно фиксировать затраты на обслуживание и следующую дату обслуживания. Отслеживание затрат на обслуживание может быть важным для бюджетирования и принятия решений о том, следует ли заменить основное средство. Вы можете прикрепить каждое основное средство к одному или нескольким страховым полисам и убедиться, что страховые премии соответствуют стоимости активов.
