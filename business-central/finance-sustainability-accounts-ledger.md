# <a name="chart-of-sustainability-accounts-and-ledger"></a>План счетов и книга экологического учета

## <a name="chart-of-sustainability-accounts"></a>План счетов экологического учета

**План счетов экологического учета** представляет собой фундаментальный структурированный список, используемый для регистрации всех данных о выбросах. Он играет роль платформы для классификации и систематизации счетов экологического учета на основе их атрибутов, таких как область выбросов или другие признаки группирования. Каждому счету обычно присваивается уникальный код или номер для удобства поиска и отслеживания, что соответствует структуре традиционного **плана счетов**, однако адаптированной именно для мониторинга связанных с устойчивым развитием данных и метрик в пределах организации. 
 
Пользователи могут добавлять **категории счетов** и **подкатегории счетов**, чтобы определить поведение системы, с выбором конкретных выбросов для отслеживания, коэффициентов выбросов, формул и тому подобных параметров.  

>[!NOTE]
>Что касается областей, в соответствии со стандартами о парниковых газах (ПГ) предусмотрено три области выбросов:  
>- **Выбросы области 1** включают выбросы, возникающие в результате стационарного и мобильного сжигания, а также непреднамеренные неконтролируемые выбросы. 
>- **Выбросы области 2** включают косвенные выбросы от производства энергии, приобретаемой у поставщиков коммунальных услуг. 
>- **Выбросы области 3** включают широкий спектр выбросов от закупленной продукции и основного оборудования; деятельности, связанной с топливом и энергией; транспортирования в верхнем и нижнем сегменте;, образующихся отходов; деловых поездок, поездок сотрудников на работу и т. д. 

Со страницы **План счетов экологического учета** можно сделать следующее:  

-   Просмотреть отчеты, в которых отображаются операции и сальдо книги экологического учета. 
-   Открыть страницу **Карточка счета экологического учета** для добавления или изменения настроек.  
-   Просмотреть категорию и подкатегорию соответствующего счета.   
-   Просмотреть отдельные сальдо по каждому выбросу для одного счета. 
-   Добавить одно или несколько **измерений** к каждому из счетов и установить фильтр по измерениям. 
    
Вы можете добавлять, изменять или удалять **счета экологического учета**. Однако во избежание расхождений удалить **счет экологического учета** нельзя, если с этим счетом связана хотя бы одна операция книги учета.  

### <a name="add-or-change-accounts"></a>Добавить или изменить счета

1. Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать 3.](media/ui-search/search_small.png "Что вы хотите сделать"), введите **План счетов экологического учета**, а затем выберите соответствующую ссылку. 
2. На странице **План счетов экологического учета** можно открыть каждый **счет экологического учета** и добавить или изменить параметры Наведите указатель на поле, чтобы увидеть короткое описание. 

Для счетов, принадлежащих к типу счетов **Итог**, нужно заполнить поле **Группировка**. Для счетов **Конец-Итог** это поле заполняется автоматически функцией создания отступа. После того как вы настроили все счета, выберите действие **Создать отступы в плане счетов экологического учета**, чтобы это сделать.  

>[!IMPORTANT]
>Если в полях **Группировка** для счетов **Сумма (до)** определения были введены до выполнения функции создания отступов, их следует ввести снова после выполнения этой функции, так как она производит запись значений поверх старых во всех полях **Сумма (до)**.  

### <a name="delete-accounts"></a>Удаление счетов

Вы можете удалить **счет экологического учета**. Однако прежде чем удалять его, вы должны убедиться, что с этим счетом не связана одна или несколько операций книги, поскольку в такой ситуации Business Central не позволит вам удалить **счет экологического учета**.  

## <a name="account-categories"></a>Категории счетов

Пользователям необходимо добавить **категорию счета экологического учета** к каждому из **счетов экологического учета**, чтобы определить поведение системы, с выбором конкретных выбросов для отслеживания, коэффициентов выбросов, формул и тому подобных параметров.  

Чтобы просмотреть **категории счетов экологического учета**, выполните следующие действия: 

1.  Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать 3.](media/ui-search/search_small.png "Что вы хотите сделать"), введите **Категории счетов экологического учета**, а затем выберите соответствующую ссылку. 
2.  На странице **Категории счетов экологического учета** можно редактировать существующий список или создавать новые категории. Для создания новой категории выберите действие **Создать**.  
3.  Заполните поля **Код** и **Описание**.   
4.  Заполните поле **Область выбросов**, выбрав одну из областей.  
5.  Выберите выбросы газа, которые вы хотите отслеживать. В настоящее время можно использовать один из вариантов: **CO2**, **CH4** или **N2O**. Вы можете выбрать любую комбинацию, которую хотите отслеживать, но для отслеживания у вас должна быть как минимум один вид выбросов.  
6.  В поле **Принцип расчета** можно выбрать любую из формул, которую вы хотите использовать, если не знаете точную величину выбросов. Здесь вы можете указать принцип расчета (формулу) для расчета выбросов. Можно выбрать один из следующих вариантов: **Топливо/электроэнергия**, **Расстояние**, **Установка** или **Произвольная**. 
7.  Если выбрана формула **Произвольная**, вы можете настроить произвольное описание в поле **Произвольное значение**.  

>[!NOTE]
>Если набора формул, предлагаемого в поле **Принцип расчета**, недостаточно, вы можете расширить это поле и добавить в систему дополнительные расчеты, которые будут использоваться в **журналах экологического учета**.  

Если вы используете **Принцип расчета** (формулы), приводится пояснение того, как система будет рассчитывать значения с использованием выбранного вами варианта (**КВ** — это **Коэффициент выбросов**, который можно настроить на странице **Подкатегория счетов экологического учета**): 

|  Тип выбросов  |  Принцип расчета  |  Формула         | Комментарий      |
|------------|--------------|------------------------------|---------------------------------|
| **ОБЛАСТЬ 1**  |
| Стационарное сжигание | Топливо/электроэнергия | Выбросы = Топливо * КВ | _где Топливо = объем топлива, потребленного котлами, нагревателями, термическими окислителями..._ |
| Мобильное сжигание | Топливо/электроэнергия | Выбросы = Топливо * КВ | _где Топливо = объем топлива, потребленного дорожными или внедорожными транспортными средствами, железнодорожным транспортом..._ |
|  |  |  Выбросы= Расстояние * КВ | _где расстояние = пробег дорожных или внедорожных транспортных средств, железнодорожного транспорта..._ |
| Неконтролируемые выбросы | Установка | Выбросы = Множитель установок * Произвольная величина / 100 * Временной коэффициент | _где Произвольная величина = потери при сборке, норма годовой утечки..._ |
| **ОБЛАСТЬ 2**  |
| Поставщики коммунальных услуг | Топливо/электроэнергия | Выбросы = Электричество * КВ | _где Топливо/Электричество = Количество электроэнергии, Количество пара, Отопление..._ |
|  | Произвольная величина | Выбросы = Произвольная величина * КВ | _где Произвольная величина = тепловая единица, тонна-час..._ |
| **ОБЛАСТЬ 3**  |
| Закупленная продукция и основное оборудование | Произвольная величина | Выбросы = Произвольная величина * КВ | _где Произвольная величина = стоимость (ГК)..._ |
| Транспортирование и распределение в верхнем сегменте | Расстояние | Выбросы= Расстояние * КВ |  |
|  | Расстояние | Выбросы= Расстояние * Множитель * КВ | _Множитель = Тонны груза_ |
| Транспортирование и распределение в нижнем сегменте | Расстояние | Выбросы= Расстояние * КВ |  |
|  | Расстояние | Выбросы= Расстояние * Множитель * КВ | _Множитель = Тонны груза_ |
| Отходы, образующиеся при производстве и утилизации проданной продукции | Произвольная величина | Выбросы = Произвольная величина * КВ | _где Произвольная величина = отходы..._ |
| Деловые поездки и поездки сотрудников на работу | Расстояние | Выбросы= Расстояние * КВ | _где Расстояние = расстояние, пройденное служебным автомобилем, арендованным автомобилем, поездом, самолетом..._ |
|  | Произвольная величина | Выбросы = Произвольная величина * КВ | _где Произвольная величина = длительность проживания в гостинице..._ |
|  | Топливо/электроэнергия | Выбросы = Топливо * КВ | _где Топливо =  объем топлива, потребленного служебным автомобилем, арендованным автомобилем..._ |

## <a name="account-subcategories"></a>Подкатегории счетов

Для каждого **счета экологического учета** пользователю необходимо указать **подкатегорию счета экологического учета**, чтобы определить коэффициенты выбросов, которые будут использоваться в формулах, однако эта подкатегория зависит от варианта отслеживания, выбранного для **категории счета экологического учета**.  

Чтобы просмотреть **подкатегории счетов экологического учета**, выполните следующие действия:  

1.  Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать 3.](media/ui-search/search_small.png "Что вы хотите сделать"), введите **Подкатегории счетов экологического учета**, а затем выберите соответствующую ссылку. 
2.  На странице **Подкатегории счетов экологического учета** можно редактировать существующий список или создавать новые категории. Для создания новой категории выберите действие **Создать**.  
3.  Заполните поля **Код** и **Описание**.   
4.  На основе выбросов газа, которые вы хотите отслеживать в **категории счетов экологического учета** и соединить с этой подкатегорией, также можно указать один или несколько коэффициентов выбросов: 

   - **Коэффициент выбросов CO2** — определяет коэффициент выбросов для выбросов CO2.  
   - **Коэффициент выбросов CH4** — определяет коэффициент выбросов для выбросов CH4. 
   - **Коэффициент выбросов N2O** — определяет коэффициент выбросов для выбросов N2O.  

5.  Если эта подкатегория связана с возобновляемой энергией, выберите поле **Возобновляемая энергия**.   

>[!NOTE]
>Поля **Импорт данных** и **Импорт из** предназначены для потенциальной интеграции с внешними системами, используемыми для сбора коэффициентов выбросов, но в **волне 1 выпуска 2024 года** использовать их как функцию по умолчанию нельзя.  

## <a name="sustainability-ledger-entries"></a>Операции книги экологического учета

**Книга экологического учета** служит для хранения всех учтенных транзакций экологического учета и систематизации всех данных о выбросах в соответствии с **планом счетов экологического учета**. Когда пользователь учитывает **журнал экологического учета**, все необходимые данные регистрируются в этой книге. Все активные отчеты формируются на основе **операций книги экологического учета**.   

Пользователь может открыть эту книгу для одного конкретного счета, используя действие **Операции книги** на странице **План счетов экологического учета**; или, чтобы открыть все операции книги, выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать 3.](media/ui-search/search_small.png "Что вы хотите сделать"), введите **Операции книги экологического учета**, а затем выберите соответствующую ссылку. Наведите указатель на поле, чтобы увидеть короткое описание.  

>[!IMPORTANT]
>После того как вы учтете данные в книге экологического учета, вы не сможете их удалить. Если вы допустили ошибку, вы можете учесть сторнирующую транзакцию с теми же данными, но с величиной с отрицательным знаком.
