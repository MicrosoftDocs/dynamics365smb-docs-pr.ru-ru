# <a name="keyboard-shortcuts"></a>Сочетания клавиш
Эта статья содержит обзор некоторых сочетаний клавиш, которые можно использовать при работе с [!INCLUDE[d365fin_long_md](includes/d365fin_long_md.md)].

## <a name="overview"></a>Обзор
Сочетания клавиш улучшают специальные возможности и могут сделать проще и эффективнее переход к другим областям и элементам на странице.

Сочетания клавиш поддерживаются большинством веб-браузеров; однако поведение может несколько отличаться.

Сочетания клавиш, описанные здесь, относятся к раскладке клавиатуры для США. Раскладка клавиш на других клавиатурах может не соответствовать клавишам на клавиатуре для США.

##  <a name="Keyboard"></a> Общие сочетания клавиш
В следующей таблице описываются сочетания клавиш для навигации и доступа к различным элементам страницы, таким как действия, раскрывающиеся списки, подстановки и т. д. Сведения о сочетаниях клавиш для навигации по записям после перехода в список см. в следующем разделе. 

|Нажмите эти клавиши|Действие|  
|----------------|-----------|  
|Tab|Перемещение фокуса на следующий элемент управления на странице, например действия, кнопки, поля или заголовки списков.| 
|Shift+Tab|Перемещение фокуса на предыдущий элемент управления на странице, например кнопки, поля или заголовки списков.|   
|ВВОД|Включение или доступ к элементу или элементу управления в фокусе.|   
|ALT+СТРЕЛКА ВНИЗ|Открытие раскрывающегося списка или поиск значения для поля.|    
|ALT + Стрелка вправо|Просмотр транзакций, которые привели к рассчитанному значению в поле.|  


## <a name="keyboard-shortcuts-in-lists"></a>Сочетания клавиш в списках

В следующей таблице описываются сочетания клавиш, которые можно использовать на странице списка. Действие сочетания клавиш несколько отличается в зависимости от представления, в котором отображается страница: представление списка или представление плиток.

Большая часть сочетаний клавиш одинаковы для операционных системы Windows и macOS; однако некоторые сочетания клавиш в macOS и Windows различаются. Они также указаны в скобках в таблице.

>[!Note]
>В следующей таблице *поле, допускающее действие,* означает поле, в котором можно выполнить какое-либо действие, например изменить значение или связать с другой страницей. В целом эти сочетания клавиш пропускают поля, в которых отображается информация, которую невозможно изменить из списка (иначе говоря, эти поля только для чтения).


|Нажмите эти клавиши<br />(в macOS)|Чтобы сделать следующее в представлении списка |Чтобы сделать следующее в представлении списка |Замечания|
|-----------------|-------|-------|-------|
|Клавиша со стрелкой вверх|Переход в этом же столбце к полю, находящемуся на строку выше.| Переход в этом же столбце к плитке, находящейся на строку выше.  |  | 
|Стрелка вниз|Переход в этом же столбце к полю, находящемуся на строку ниже. |Переход в этом же столбце к плитке, находящейся на строку ниже. | |
|Клавиша со стрелкой вправо|Переход в этой же строке к следующему полю, допускающему действие, справа.| Переход в этой же строке к следующей плитке справа. |Это не относится к случаю, когда список допускает изменение.|
|Стрелка влево|Переход в этой же строке к предыдущему полю слева. | Переход в этой же строке к предыдущей плитке слева. |Это не относится к случаю, когда список допускает изменение.|
|Дом<br />(Fn+Стрелка влево)|Переход к первому полю в строке.|Переход к первой плитке в строке.||
|End<br />(Fn+Стрелка вправо)|Переход к последнему полю, допускающему действие, в строке.|Переход к последней плитке в строке.||
|Page Up<br />(Fn+Стрелка вверх)|Переход в этом же столбце к самой верхней видимой строке.|Переход в этом же столбце к самой верхней видимой строке.||
|Page Down<br />(Fn+Стрелка вниз)|Переход в этом же столбце к самой нижней видимой строке.|Переход в этом же столбце к самой нижней видимой строке.||
|ВВОД<br />(Fn+Стрелка вниз)|Открытие записи, связанной с этим полем|Открытие записи.| Действует только в случае, если страница карты связана с записью.|
|Ctrl+Home<br />(Fn+Ctrl+Стрелка влево)|Выбор первой строки в списке; фокус перемещается на первое поле, допускающее действие, в строке|Переход к плитке в первой строке в списке||
|Ctrl+End<br />(Fn+Ctrl+Стрелка вправо)|Выбор последней строки в списке; фокус перемещается на первое поле, допускающее действие, в строке|Переход к первой плитке в последней строке в списке||
|Ctrl+Щелчок<br />(Cmd+Щелчок)|Расширение выбора строк для включения строки, на которой вы щелкнули.|Неприменимо||
|Shift+Щелчок|Расширение выбора строк для включения строки, на которой вы щелкнули, и всех расположенных между ними строк.|Неприменимо|Это можно использовать после использования "Ctrl+Стрелка вверх" или "Ctrl+Стрелка вниз" для расширения выбора.|
|Ctrl+стрелка вверх<br />(Ctrl+Cmd+Стрелка вверх)|Перемещение фокуса на строку выше и сохранение выбора текущей строки.|Неприменимо||
|Ctrl+стрелка вниз<br />(Ctrl+Cmd+Стрелка вниз)|Перемещение фокуса на строку ниже и сохранение выбора текущей строки.|Неприменимо||
|Ctrl+Пробел<br />(Ctrl+Cmd+Пробел)|Расширение выбора строк для включения строки, на которой находится фокус.|Неприменимо|Это можно использовать после использования "Ctrl+Стрелка вверх" или "Ctrl+Стрелка вниз" для расширения выбора.|
|Ctrl+A|Выбор всех строк.|Неприменимо||
|Shift+Стрелка вверх|Расширение выбора строк для включения расположенной выше строки.|Неприменимо||
|Shift+Стрелка вниз|Расширение выбора строк для включения расположенной ниже строки.|Неприменимо||
|Shift+Page Up<br />(Shift+Fn+Стрелка вверх)|Расширение выбора стирок для включения всех видимых строк над текущим выбором строк.|Неприменимо||
|Shift+Page Down<br />(Shift+Fn+Стрелка вниз)|Расширение выбора стирок для включения всех видимых строк под текущими выбранными строками.|Неприменимо||
|Ctrl+Enter|Перемещение фокуса на следующий элемент вне списка.|Перемещение фокуса на следующий элемент вне списка.||
