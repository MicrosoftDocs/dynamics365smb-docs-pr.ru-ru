# <a name="keyboard-shortcuts"></a>Сочетания клавиш
Эта статья содержит обзор некоторых сочетаний клавиш, которые можно использовать при работе с [!INCLUDE[prodshort](includes/prodshort.md)].

[Таблица сочетаний клавиш в удобном для печати виде](keyboard-shortcuts-cheatsheet.md)

## <a name="overview"></a>Обзор
Сочетания клавиш улучшают специальные возможности и могут сделать проще и эффективнее переход к другим областям и элементам на странице.

Сочетания клавиш поддерживаются большинством веб-браузеров, однако поведение может несколько отличаться.

Сочетания клавиш, описанные здесь, относятся к раскладке клавиатуры для США. Раскладка клавиш на других клавиатурах может не соответствовать клавишам на клавиатуре для США.

Большая часть сочетаний клавиш одинаковы для операционных системы Windows и macOS; однако некоторые сочетания клавиш в macOS отличаются. В таблицах в следующих разделах такие сочетания клавиш указаны в скобках.

##  <a name="Keyboard"></a> Общие сочетания клавиш
В следующей таблице описываются сочетания клавиш для навигации и доступа к различным элементам страницы, таким как действия, раскрывающиеся списки, подстановки и т. д. Сведения о сочетаниях клавиш для навигации по записям после перехода в список см. в следующем разделе.

|Нажмите эти клавиши<br />(в macOS)|Действие|  
|----------------|-----------|  
|Alt+Q<br />(Ctrl+Alt+Q)|Открытие поля **Что вы хотите сделать**, которое может помочь найти страницу, отчет или действие на текущей странице либо статью в документации.|
|Alt+Стрелка вверх|Отображение всплывающей подсказки для поля или заголовка столбца таблицы. Если поле содержит ошибки проверки, нажмите кнопки "Alt+Стрелка вверх" для отображения ошибки проверки. Нажмите кнопки "Esc" или "Alt+Стрелка вверх" для закрытия всплывающей подсказки.|
|Tab|Перемещение фокуса на следующий элемент управления на странице, например действия, кнопки, поля или заголовки списков.|
|Shift+Tab|Перемещение фокуса на предыдущий элемент управления на странице, например действия, кнопки, поля или заголовки списков.|   
|ВВОД|Включение или доступ к элементу или элементу управления в фокусе.|   
|ALT+СТРЕЛКА ВНИЗ|Открытие раскрывающегося списка или поиск значения для поля.|    
|ALT + Стрелка вправо|Просмотр транзакций, которые привели к рассчитанному значению в поле.|  
|F5|Обновление данных на текущей странице.|Используйте для обеспечения, чтобы данные на странице были актуальными с учетом всех изменений, сделанных другими пользователями во время вашей работы.|
|Ctrl+F5|Перезагрузка приложения [!INCLUDE[prodshort](includes/prodshort.md)].|Это аналогично выбору обновлении или перезагрузки в браузере.|
|Esc|Закрытие текущей страницы или раскрывающегося списка.|

|Ctrl+Enter|Перемещение фокуса на следующий элемент вне списка.|Перемещение фокуса на следующий элемент вне списка.||

### <a name="KeyboardFilter"></a> Сочетание клавиш для поиска в списках и фильтрации списков

|Нажмите эти клавиши<br />(в macOS)|Действие|
|-----------------|-------|
|F3|Переключение поля поиска.<ul><li>Включает поле поиска, чтобы можно было ввести текст для поиска.</li><li>Если поле поиска уже активизировано, F3 возвращает список без очистки текста поиска.</li><ul>|  
|Shift+F3|Открывает и закрывает область фильтров.<ul><li> Если область фильтров не открыта, Shift+F3 открывает ее и фокусируется на действии **+ Фильтр** в разделе **Фильтровать список по**, что позволяет просто нажать ВВОД, чтобы начать добавлять фильтр по полю.</li><li>Если область фильтров уже открыта, Shift+F3 закрывает ее, но не очищает никакие фильтры, которые уже добавлены.</li></ul>|
|Ctrl+Shift+F3|Открывает и закрывает область фильтров.<ul><li> Если область фильтров не открыта, Ctrl+Shift+F3 открывает ее и фокусируется на действии **+ Фильтр** в разделе **Фильтровать итоги по**, что позволяет просто нажать ВВОД, чтобы начать добавлять фильтр по итогам.</li><li>Если область фильтров уже открыта, Ctrl+Shift+F3 закрывает ее, но не очищает никакие фильтры, которые уже добавлены.</li></ul>  |
|Alt+F3|Переключение фильтрации на это значение.<ul><li>Применение фильтра столбца по выбранному значению поля в списке. Это то же самое, что и выбор **Фильтровать до этого значения** из заголовка столбцов. Открывается область фильтров, задает фильтр на выбранное значение, а фокус остается в ячейке в списке.</li><li>Если столбец уже отфильтрован, Alt+F3 очищает фильтр в этом столбце.</li></ul> |
|Shift+Alt+F3|Открывается область фильтров и добавляется фильтр в выбранном столбце в списке. Фокус находится на новом поле фильтра, что позволяет сразу же начать вводить критерии фильтра.<br /><br /> Это делает то же, что и выбор пункта **Фильтр** выбор столбцов. Меню er. Отображает область фильтров, добавляет фильтр, задает фокус на него, чтобы пользователь мог ввести значение для фильтрации по нему.<br /><br />Если уже есть фильтр в поле, новый фильтр добавляется. |
|Ctrl+Shift+Alt+F3|Сброс фильтров. Выполняется то же действие, что и при выборе **Сброс фильтров** в области фильтров, и применяется к фильтрам по полям и итогам.<br /><br /> Фильтры возвращаются к фильтрам по умолчанию для текущего представления. Если текущее представление **Все**, это эквивалентно возврату к нефильтрованному представлению со всеми записями. |
|Ctrl+Enter|Возврат к списку из области фильтров.|

## <a name="a-namecalendarshortcuts-keyboard-shortcuts-in-the-calendar-date-picker"></a><a name="calendarshortcuts"/> Сочетания клавиш в календаре (элементе управления для выбора даты)
При задании значения в поле даты можно либо ввести дату вручную, либо открыть календарь (элемент управления выбора даты), который позволяет выбрать нужную дату. В следующей таблице приведены сочетания клавиш для календаря.

|Нажмите эти клавиши<br />(в macOS)|Действие|
|-----------------|-------|
|Page Up|Переход к предыдущему месяцу.|  
|Page Down|Переход к следующему месяцу.|
|Ctrl+Home|Открытие календаря, если он закрыт.|
|Ctrl+Home<br />(Cmd+Home)|Переход к текущему месяцу, текущему дню.|
|Ctrl+стрелка влево<br />(Cmd+стрелка влево)|Переход к предыдущему дню.|
|Ctrl+стрелка вправо<br />(Cmd+стрелка вправо)|Переход к следующему дню.|
|Ctrl+стрелка вверх<br />(Cmd+стрелка вверх)|Переход к предыдущей неделе, тому же дню недели.|
|Ctrl+стрелка вниз<br />(Cmd+стрелка вниз)|Переход к следующей неделе, тому же дню недели.|
|ВВОД|Выбор даты, на которой установлен фокус.|
|Ctrl+End<br />(Cmd+End)|Закрыть календарь и удалить текущую дату.|
|Esc|Закрыть календарь без выбора, оставив текущую дату.|
