# <a name="business-central-data-replication-extension"></a>Расширение репликации данных Business Central**

Это расширение проводит вас через процесс настройки среды интеллектуального облака.  После настройки вашей среды интеллектуального облака можно будет реплицировать данные из вашего локального решения [!INCLUDE[prodshort](includes/prodshort.md)] в ваш облачный клиент [!INCLUDE[prodshort](includes/prodshort.md)].  Это позволит вам полностью использовать преимущества облака для вашего бизнеса, такие как улучшенный анализ бизнеса, искусственный интеллект, доступ с нескольких устройств и доступ в любой момент из любого места.

Это расширение должно использоваться совместно с расширением **Интеллектуальное облако Business Central**.
