В финансовом журнале отображается только ограниченное число полей в строке журнала по умолчанию. Если необходимо видеть дополнительные поля, например поле **Согласование общей товарной группы***, выберите действие **Страница** на навигационной панели, затем выберите **Показать больше столбцов**. Чтобы снова спрятать дополнительные поля, выберите **Показать меньше столбцов**.
