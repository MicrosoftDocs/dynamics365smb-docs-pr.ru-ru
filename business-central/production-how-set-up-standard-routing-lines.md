## <a name="to-set-up-a-standard-task"></a><span data-ttu-id="13826-106">Чтобы настроить стандартную задачу</span><span class="sxs-lookup"><span data-stu-id="13826-106">To set up a standard task</span></span>

1. <span data-ttu-id="13826-107">Выберите значок ![Лампочка, которая открывает функцию "Что вы хотите сделать"](media/ui-search/search_small.png "Что вы хотите сделать"), введите **Стандартные задачи**, затем выберите соответствующую ссылку.</span><span class="sxs-lookup"><span data-stu-id="13826-107">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Standard Tasks**, and then choose the related link.</span></span>
2. <span data-ttu-id="13826-108">Выберите действие **Создать**.</span><span class="sxs-lookup"><span data-stu-id="13826-108">Choose the **New** action.</span></span>
3. <span data-ttu-id="13826-109">Заполните соответствующим образом поля.</span><span class="sxs-lookup"><span data-stu-id="13826-109">Fill in the fields as necessary.</span></span> [!INCLUDE[tooltip-inline-tip](includes/tooltip-inline-tip_md.md)]
4. <span data-ttu-id="13826-110">Выбрать **Инструменты**, **персонал**, **Описание**, или действие **Инструменты**, а затем на открывшейся странице добавьте соответствующую запись.</span><span class="sxs-lookup"><span data-stu-id="13826-110">Choose the **Tools**, **Personnel**, **Description**, or **Tools** action, and then, on the page that opens, add the relevant entry.</span></span>

## <a name="to-add-a-standard-task-to-an-operation"></a><span data-ttu-id="13826-111">Добавление стандартной задачи к операции</span><span class="sxs-lookup"><span data-stu-id="13826-111">To add a standard task to an operation</span></span>

1. <span data-ttu-id="13826-112">Выберите значок ![Лампочка, которая открывает функцию "Что вы хотите сделать"](media/ui-search/search_small.png "Что вы хотите сделать"), введите **Маршруты**, затем выберите соответствующую ссылку.</span><span class="sxs-lookup"><span data-stu-id="13826-112">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Routings**, and then choose the related link.</span></span>
2. <span data-ttu-id="13826-113">Откройте соответствующий маршрут.</span><span class="sxs-lookup"><span data-stu-id="13826-113">Open the relevant routing.</span></span>
3. <span data-ttu-id="13826-114">Выберите нужную строку маршрута, выберите действие **Операции**, затем выберите действие **Инструменты**, **Персонал**, **Описание** или **Инструменты**.</span><span class="sxs-lookup"><span data-stu-id="13826-114">Select a relevant routing line, choose the **Operations** action, and then choose the **Tools**, **Personnel**, **Description**, or **Tools** action.</span></span>

<span data-ttu-id="13826-115">При печати листа маршрутизации вы можете включить элементы из стандартной задачи, которые вы добавили в строках маршрута.</span><span class="sxs-lookup"><span data-stu-id="13826-115">When you print the routing sheeting, you can choose to include the elements from the standard task that you added to the routing lines.</span></span>
