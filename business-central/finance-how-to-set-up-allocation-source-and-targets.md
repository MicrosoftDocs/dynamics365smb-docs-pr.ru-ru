## <a name="to-set-up-cost-allocations"></a><span data-ttu-id="84dc8-109">Настройка распределения затрат</span><span class="sxs-lookup"><span data-stu-id="84dc8-109">To set up cost allocations</span></span>  
1.  <span data-ttu-id="84dc8-110">Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать](media/ui-search/search_small.png "Что вы хотите сделать"), введите **Распределение затрат**, затем выберите связанную ссылку.</span><span class="sxs-lookup"><span data-stu-id="84dc8-110">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Cost Allocation**, and then chose the related link.</span></span>  
2.  <span data-ttu-id="84dc8-111">На странице **Распределение затрат** выберите действие **Правка**.</span><span class="sxs-lookup"><span data-stu-id="84dc8-111">On the **Cost Allocation** page, choose the **Edit** action.</span></span>  
3.  <span data-ttu-id="84dc8-112">Введите идентификатор для источника распределения в поле **ИД**.</span><span class="sxs-lookup"><span data-stu-id="84dc8-112">Enter an ID for the allocation source in the **ID** field.</span></span>  
4.  <span data-ttu-id="84dc8-113">Определите уровень в виде числа между 1 и 99 в поле **Уровень**.</span><span class="sxs-lookup"><span data-stu-id="84dc8-113">Define a level as a number between 1 and 99 in the **Level** field.</span></span> <span data-ttu-id="84dc8-114">Учет распределения будет соответствовать порядку следования уровней.</span><span class="sxs-lookup"><span data-stu-id="84dc8-114">The allocation posting will follow the order of the levels.</span></span>  
5.  <span data-ttu-id="84dc8-115">Введите тип затрат, чтобы определить, какие типы затрат будут распределены, в поле **Диапазон типов затрат**.</span><span class="sxs-lookup"><span data-stu-id="84dc8-115">Enter a cost type to define which cost types will be allocated in the **Cost Type Range** field.</span></span> <span data-ttu-id="84dc8-116">Если все себестоимости типа затрат распределены, значит диапазон не определен.</span><span class="sxs-lookup"><span data-stu-id="84dc8-116">If all costs for a cost type are allocated, no range is defined.</span></span>  
6.  <span data-ttu-id="84dc8-117">Введите центр затрат вместе с распределяемыми затратами в поле **Код центра затрат**.</span><span class="sxs-lookup"><span data-stu-id="84dc8-117">Enter a cost center together with costs to be allocated in the **Cost Center Code** field.</span></span>  
7.  <span data-ttu-id="84dc8-118">Введите объект затрат вместе с распределяемыми затратами в поле **Код объекта затрат**.</span><span class="sxs-lookup"><span data-stu-id="84dc8-118">Enter a cost object together with costs to be allocated in the **Cost Object Code** field.</span></span> <span data-ttu-id="84dc8-119">Чаще всего это поле остается пустым, поскольку объекты затрат редко распределяются в другие объекты затрат.</span><span class="sxs-lookup"><span data-stu-id="84dc8-119">Most often, this field stays empty, because cost objects are rarely allocated to other cost objects.</span></span>  
8.  <span data-ttu-id="84dc8-120">Введите тип затрат в поле **Кредит по типу затрат**.</span><span class="sxs-lookup"><span data-stu-id="84dc8-120">Enter a cost type in the **Credit to Cost Type** field.</span></span> <span data-ttu-id="84dc8-121">Распределяемые затраты будут кредитоваться в исходный тип затрат.</span><span class="sxs-lookup"><span data-stu-id="84dc8-121">The costs that are allocated will be credited to the source cost type.</span></span> <span data-ttu-id="84dc8-122">Учет кредита будет осуществляться по указанному здесь типу затрат.</span><span class="sxs-lookup"><span data-stu-id="84dc8-122">The credit posting will be posted to the cost type given here.</span></span>  
9. <span data-ttu-id="84dc8-123">Определите цели распределения на экспресс-вкладке **Строки**.</span><span class="sxs-lookup"><span data-stu-id="84dc8-123">On the **Lines** FastTab, define the allocation targets.</span></span> <span data-ttu-id="84dc8-124">В первой строке введите тип затрат в поле **Тип назначения затрат**.</span><span class="sxs-lookup"><span data-stu-id="84dc8-124">On the first line, enter a cost type in the **Target Cost Type** field.</span></span> <span data-ttu-id="84dc8-125">Это определяет, по какому типу затрат распределение подлежит дебету.</span><span class="sxs-lookup"><span data-stu-id="84dc8-125">It defines which cost type the allocation is debited to.</span></span>  
10. <span data-ttu-id="84dc8-126">В первой строке введите первую цель распределения в поле **Целевой центр затрат** или **Целевой объект затрат**.</span><span class="sxs-lookup"><span data-stu-id="84dc8-126">On the first line, enter the first allocation target in the **Target Cost Center** field or **Target Cost Object** the field.</span></span> <span data-ttu-id="84dc8-127">Эти два поля определяют, распределение по какому месту возникновения затрат или объекту затрат подлежит дебету.</span><span class="sxs-lookup"><span data-stu-id="84dc8-127">These two fields define which cost center or cost object the allocation is debited to.</span></span> <span data-ttu-id="84dc8-128">Заполнить можно только одно из этих полей, но не оба.</span><span class="sxs-lookup"><span data-stu-id="84dc8-128">You can only fill in one of these fields, but not both.</span></span>  
11. <span data-ttu-id="84dc8-129">Повторите эти же действия на второй строке для настройки дополнительных целей распределения.</span><span class="sxs-lookup"><span data-stu-id="84dc8-129">Repeat the same steps on the second line to set up additional allocation targets.</span></span>  
12. <span data-ttu-id="84dc8-130">После настройки цели и источников распределения выберите действие **Вычислить ключ распределения** для вычисления полной стоимости акций.</span><span class="sxs-lookup"><span data-stu-id="84dc8-130">After you have set up the allocation target and sources, choose the **Calculate Allocation Key** action to calculate the total share values.</span></span>  

> [!NOTE]  
>  <span data-ttu-id="84dc8-131">Установите флажок **Заблокировано**, чтобы отключить настройку распределения.</span><span class="sxs-lookup"><span data-stu-id="84dc8-131">Select the **Blocked** check box to deactivate the allocation setup.</span></span>
