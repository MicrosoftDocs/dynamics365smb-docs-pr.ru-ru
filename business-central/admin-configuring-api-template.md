<span data-ttu-id="fad22-117">Если для API (ИД страницы) выбрано несколько шаблонов, шаблоны применяются в порядке, определенном в поле **Порядок**.</span><span class="sxs-lookup"><span data-stu-id="fad22-117">If there is more than one template selected for an API (Page ID), the templates are applied in the order defined in the **Order** column.</span></span>

3. <span data-ttu-id="fad22-119">Выберите значение в поле **ИД страницы**.</span><span class="sxs-lookup"><span data-stu-id="fad22-119">Select a **Page ID** value.</span></span>  
<span data-ttu-id="fad22-120">Это страница для API, к которому будет применяться шаблон.</span><span class="sxs-lookup"><span data-stu-id="fad22-120">This is the page for the API to which the template will be applied.</span></span> <span data-ttu-id="fad22-121">Поиск в поле **ИД страницы** выводит список всех API-интерфейсов, имеющихся в библиотеке.</span><span class="sxs-lookup"><span data-stu-id="fad22-121">The **Page ID** lookup provides a list of all APIs available in the library.</span></span>
4. <span data-ttu-id="fad22-122">Выберите значение в поле **Код шаблона**.</span><span class="sxs-lookup"><span data-stu-id="fad22-122">Select a value in the **Template Code** field.</span></span>  
<span data-ttu-id="fad22-123">Код шаблона — это код для шаблона, который был определен на странице **Шаблоны конфигураций**.</span><span class="sxs-lookup"><span data-stu-id="fad22-123">The template code is the code for the template that was defined on the **Configuration Templates** page.</span></span> <span data-ttu-id="fad22-124">Определенные в шаблоне значения применяются к API.</span><span class="sxs-lookup"><span data-stu-id="fad22-124">The template values defined are applied to the API.</span></span> 
5. <span data-ttu-id="fad22-125">В поле **Условия** укажите, какой шаблон должен быть применен.</span><span class="sxs-lookup"><span data-stu-id="fad22-125">In the **Conditions** field, specify which template should be applied.</span></span>  
<span data-ttu-id="fad22-126">Заданный шаблон применяется к новой записи, создаваемой через API, если и только если значения, уже определенные для нового экземпляра записи, соответствуют условиям, определенным в поле **Условия**.</span><span class="sxs-lookup"><span data-stu-id="fad22-126">The defined template is applied to a new record created through the API if, and only if, the conditions defined in the **Conditions** field are met by the values already defined for the new instance of the entity.</span></span>
