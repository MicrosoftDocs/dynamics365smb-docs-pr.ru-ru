# Настройка разноски внутрихолдинговых транзакций

Межфирменные проводки внутри компании упрощают учет для двух или более компаний для централизованного финансового отдела и бухгалтеров в компаниях-межфирменных партнерах. Чтобы отправить транзакцию (такую как строка журнала продаж) из одной организации и обеспечить автоматическое создание соответствующей транзакции (такой как строка журнала покупок) в партнерской организации, вовлеченные в этот процесс организации должны согласовать общий план счетов и настроить измерения для использования в межфирменных транзакциях. В качестве межфирменного плана счетов может использоваться, например, упрощенная версия плана счетов родительской организации. Каждая организация сопоставляет свой полный план счетов с общим межфирменным планом счетов, а свои измерения — с межфирменными измерениям.  

Необходимо также настроить код межфирменного партнера для каждой организации [!INCLUDE [prod_short](includes/prod_short.md)], который согласован всеми компаниями и назначить его карточкам клиентов и поставщиков соответственно.  

При создании или получении межфирменных строк с товарами можно либо использовать собственные номера товаров, либо настроить для каждого соответствующего товара номера, присвоенные партнером, или в поле **Код товара поставщика**, или в поле **Код общего товара** на карточке товара. Можно также использовать функцию **Ссылка товара**: чтобы сопоставить ваши номера товаров с описаниями товаров ваших межфирменных партнеров, откройте карточку каждого товара, затем выберите действие **Ссылки** для настройки ссылок между вашими описаниями товара и описаниями межфирменного партнера. Дополнительные сведения см. в разделе [Использование ссылок по товару](inventory-how-use-item-cross-refs.md).

При создании транзакций межфирменных продаж, включающих ресурсы, для каждого соответствующего ресурса необходимо заполнить поле **Партнер МФ - номер счета ГК покупки** на карточке ресурса. Это номер межфирменного счета главной книги в организации-партнере, в котором учитывается сумма для данного ресурса. Дополнительные сведения см. в разделе [Настройка ресурсов](projects-how-setup-resources.md).

> [!NOTE]
> Межфирменные транзакции покупки, включающие ресурсы, основные средства и товарные издержки, поддерживаются не полностью. В компании вашего межфирменного партнера поле **Тип строки** будет пустым в строках документа покупки, которые включают эти объекты. Необходимо выполнить обновление поля вручную.

## Автоматическое принятие транзакций от межфирменных партнеров

В первой волне выпуска 2022 года представлена новая страница **Межфирменная настройка**, которая может ускорить обработку транзакций от ваших межфирменных партнеров. На этой странице можно указать, будет ли эта компания автоматически создавать строки журнала на основе записей межфирменного партнера со страницы **Общий журнал МФ**. Строки журнала создаются для вас, но не учитываются. Вы можете использовать следующие поля на новой странице Межфирменная настройка, чтобы указать, где создавать полученные транзакции межфирменного журнала:

* **Шаблон фин. жур. МФ по умолч.**
* **Раздел фин. жур. МФ по умолч.**

> [!NOTE]
> Если ваша организация использовала межфирменные функции в [!INCLUDE [prod_short](includes/prod_short.md)] до волны 1 выпуска 2022 для автоматического принятия транзакций, ваш администратор должен включить переключатель функции **Автоматически принимать транзакции межфирменного общего журнала** на странице **Управление функциями**.

1. Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок введите **Межфирменные партнеры**, а затем выберите связанную ссылку.

Поэтому, если вы выберете **Расположение файла** в поле **Тип передачи**, поле **Путь к папке** недоступно. Вместо этого файл будет загружен в папку "Загрузки" на вашем компьютере. Затем вы отправляете файл кому-то в партнерской компании, например, по электронной почте. Для более прямого процесса мы рекомендуем вам выбрать вместо этого пункт **Электронная почта**.

> [!NOTE]
> Для межфирменной проводки, если вы включили выключатель **Авто. Поэтому важно иметь бизнес-процесс для управления дубликатами. Например, путем удаления таких заказов на покупку при получении счета на покупку от внутрихолдингового партнера.


## Настройка межфирменных поставщиков и межфирменных клиентов

1. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **Поставщики**, а затем выберите связанную ссылку.
2. Кроме того, доступ к поставщику можно получить из поля **Код поставщика** на странице **Межфирменный партнер**.
3. Откройте карточку для поставщика, который является межфирменным партнером. Дополнительные сведения см. в разделе [Регистрация новых поставщиков](purchasing-how-register-new-vendors.md).
4. В поле **Код межфирменного партнера** выберите соответствующий код межфирменного партнера.
5. Повторите шаги с 1 по 4 для клиентов.

## Настройка межфирменного плана счетов

Чтобы группа организаций могла осуществлять межфирменные транзакции, они должны согласовать план счетов, который будет использоваться как общий эталон. Необходимо согласовать с организациями-партнерами номера счетов, которые вы все будете использовать при создании межфирменных транзакций. Например, родительская организация группы создает упрощенную версию собственного плана счетов, затем экспортирует его в файл XML и распространяет этот файл среди всех организаций группы.  

Если план счетов вашей компании определяет межфирменный план счетов для ваших компаний-партнеров, следуйте процессу, описанному в разделе [Настройка определяющего межфирменного плана счетов](intercompany-how-setup.md#to-set-up-the-intercompany-chart-of-accounts).  

Если ваша организация является дочерней организацией и получила файл XML, содержащий общий межфирменный план счетов, который ваша группа будет использовать как общий эталон, следуйте процедуре [Импорт межфирменного плана счетов](intercompany-how-setup.md#to-import-the-intercompany-chart-of-accounts).  

### Настройка межфирменного плана счетов

1. Выберите значок ![Лампочка, которая открывает функцию «Что вы хотите сделать»](media/ui-search/search_small.png "Что вы хотите сделать"), значок, введите **План счетов межфирменного учета**, а затем выберите связанную ссылку.
2. На странице **План счетов межфирменного учета** введите каждый счет в своей строке страницы.  
3. Если ваш межфирменный план счетов совпадает с обычным планом счетов или похож на него, можно автоматически заполнить страницу, выбрав действие **Копировать из плана счетов**. Новые строки при необходимости можно изменить.

### Экспорт межфирменного плана счетов

Чтобы межфирменные партнеры могли импортировать определяющий план счетов, необходимо экспортировать его в файл.

1. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **План счетов межфирменного учета**, а затем выберите связанную ссылку.
2. На странице **План счетов межфирменного учета** выберите действие **Экспорт**, затем выберите кнопку **Сохранить**.
3. Укажите имя и место сохранения файла XML, затем выберите кнопку **Сохранить**.  

### Импорт межфирменного плана счетов  

Когда существует файл для определения межфирменного плана счетов, межфирменные партнеры могут импортировать его, чтобы у них гарантировано были те же самые счета.  

1. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **План счетов межфирменного учета**, а затем выберите связанную ссылку.  
2. На странице **План счетов межфирменного учета** выберите действие **Импорт**.  
3. Выберите имя и местонахождение файла XML, затем выберите кнопку **Открыть**.  

Страница **МФ - план счетов** заполняется новыми или измененными строками счетов ГК в соответствии с межфирменным планом счетов в файле. Все существующие независимые строки на странице остаются без изменений.

### Сопоставление межфирменного плана счетов с планом счетов вашей организации  

После определения или импорта межфирменного плана счетов, согласованного для использования вами и вашими партнерами, необходимо связать каждый из межфирменных счетов ГК с одним из счетов ГК своей организации. На странице **МФ - план счетов** укажите, как межфирменные счета ГК во входящих транзакциях будут преобразовываться в счета главной книги плана счетов вашей организации.

Если номера счетов межфирменного плана счетов совпадают с номерами соответствующих счетов плана счетов, то сопоставить счета можно автоматически.

1. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **План счетов межфирменного учета**, а затем выберите связанную ссылку.  
2. Выберите строки, которые требуется соотнести автоматически, затем выберите действие **Соотнести со счетом с тем же номером**.  
3. Для каждого межфирменного счета в главной книге, который не был сопоставлен автоматически, заполните поле **Соотнесение номера счета ГК**.  

## Настройка счетов главной книги по умолчанию для МФ партнеров  

При создании строки межфирменной продажи или покупки для отправки в качестве исходящей транзакции необходимо ввести счет из межфирменного плана счетов в качестве значения по умолчанию, указывающего, на каком счету партнерской организации учитывается сумма. На странице **План счетов**, для счетов, которые часто используются в строках исходящего межфирменного журнала, можно указать счет ГК по умолчанию для межфирменного партнера. Например, для своих дебиторских счетов можно ввести соответствующие счета к оплате из межфирменного плана счетов.  

Затем, когда вы ввели счет главной книги в поле **Номер баланс. счета** строки межфирменного журнала со значением **Межфирменный партнер** в поле **Тип счета**, поле **Партнер МФ - счет ГК** заполняется автоматически.  

1. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **План счетов**, а затем выберите связанную ссылку.  
2. В строке для счета ГК, который используется для межфирменных транзакций, в поле **Счет ГК по умолчанию для МФ партнера**, укажите межфирменный счет главной книги, на котором ваш партнер будет выполнять учет, когда вы производите учет на счет главной книги в строке.  
3. Повторите шаг 2 для каждого счета, который часто вводится в поле **Номер баланс. счета** строки межфирменного журнала или документа.

## Настройка межфирменных измерений

Если вам и вашим партнерам по межфирменным взаимодействиям нужна возможность обмениваться транзакциями и связанными с ними измерениями, необходимо согласовать измерения, которые будут использоваться всеми партнерами. Например, родительская организация группы создает упрощенную версию собственного набора измерений, затем экспортирует их в файл XML и распространяет этот файл среди всех организаций группы. Затем каждый из филиалов импортирует этот файл XML на страницу **Межфирменные измерения** и сопоставляет межфирменные измерения с измерениями на своей собственной странице **Измерения**.  

> [!NOTE]
> Каждая компания в [!INCLUDE [prod_short](includes/prod_short.md)] должна сопоставлять измерения с межфирменными измерениями для исходящих документов и сопоставлять межфирменные измерения с собственными измерениями для входящих документов. Это сопоставление помогает обеспечить согласованность между компаниями. Дополнительные сведения см. в разделе [Сопоставление межфирменных измерений с измерениями собственной организации](#to-map-intercompany-dimensions-to-your-companys-dimensions).

Если ваша организация является материнской организацией и определила набор межфирменных измерений, который ваша группа будет использовать как общий эталон, следуйте процедуре [Определение межфирменных измерений](intercompany-how-setup.md#to-define-the-intercompany-dimensions).

Если ваша организация является дочерней и получила файл XML, содержащий межфирменные измерения, которые ваша группа будет использовать как общий эталон, следуйте процедуре [Импорт межфирменных измерений](intercompany-how-setup.md#to-import-the-intercompany-dimensions).

### Определение межфирменных измерений

1. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок введите **Межфирменные измерения**, а затем выберите связанную ссылку.  
2. На странице **Межфирменные измерения** введите каждое измерение в своей строке страницы.

    Если ваши межфирменные измерения совпадают с измерениями вашей организации или похожи на них, можно заполнить страницу автоматически, используя функцию **Копировать из измерений**, а затем отредактировать полученные строки.  
3. Для экспорта межфирменных измерений в XML-файл для передачи организациям-партнерам, выберите действие **Экспорт**.  
4. Укажите имя и место сохранения файла XML, затем выберите кнопку **Сохранить**.  

### Импорт межфирменных измерений  

Когда существует файл для определения межфирменных измерений, межфирменные партнеры могут импортировать его, чтобы у них гарантировано были те же самые измерения.  

1. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок введите **Межфирменные измерения**, а затем выберите связанную ссылку.  
2. На странице **Межфирменные измерения** выберите действие **Импорт**.  
3. Укажите имя и местонахождение файла XML, затем выберите кнопку **Открыть**.  

Импортируются строки на страницах **Межфирменные измерения** и **Значения межфирменных измерений**.  

### Сопоставление межфирменных измерений с измерениями собственной организации

После определения или импорта измерений, которые вы и ваши МФ-партнеры согласились использовать, необходимо связать каждое из межфирменных измерений с одним из измерений вашей организации и наоборот. На странице **Межфирменные измерения** укажите, как межфирменные измерения *входящих транзакций* будут преобразовываться в измерения из списка измерений вашей организации. На странице **Измерения** укажите, как ваши измерения будут преобразовываться в межфирменные измерения для *исходящих транзакций*.

Если код каких-либо из межфирменных измерений совпадает с кодом соответствующих измерений в списке измерений вашей организации, приложение может сопоставлять измерения автоматически, после чего вы можете автоматически сопоставить счета.  

На следующих шагах вы сначала сопоставляете внутрихолдинговые измерения с измерениями для входящих документов на странице **Межфирменные измерения**. Затем вы сопоставляете измерения с межфирменными измерениями для исходящих документов на странице **Измерения**.

1. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок введите **Межфирменные измерения**, а затем выберите связанную ссылку.
2. На странице **Межфирменные измерения** выберите строки, которые требуется соотнести автоматически, затем выберите действие **Соотнести с изм. с тем же кодом**.
3. Для каждого межфирменного измерения, которое не было отображено автоматически, заполните поле **Соотнесение кода измерения**.

    Возможно, вам придется добавить поле к вашему представлению. Дополнительные сведения см. в разделе [Персонализация рабочей области](ui-personalization-user.md).
4. Выберите действие **Значения межфирменных измерений**.
5. На странице **Значения межфирменных измерений** заполните поле **Соотнесение МФ - код значения измерения**.

    Продолжайте сопоставлять измерения с межфирменными измерениями, выполняя аналогичные шаги.
6. Выберите ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок введите **Измерения**, а затем выберите связанную ссылку.
7. На странице **Измерения** выберите строки, которые требуется соотнести автоматически, затем выберите действие **Соотнести с изм. МФ с тем же кодом**.
8. Для каждого межфирменного измерения, которое не было отображено автоматически, заполните поле **Соответствие коду значения МФ-измерения**.
9. Выберите действие **Значения измерения**.
10. На странице **Значения измерений** заполните поле **Соответствие коду значения МФ-измерения**.
