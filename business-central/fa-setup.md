# <a name="setting-up-fixed-assets"></a>Настройка основных средств

До начала работы с основными средствами (ОС) необходимо определить несколько вещей.  

* Как амортизируются основные средства.  
* Способ регистрации затрат на приобретение, амортизаций и других значений в главной книге.  
* По желанию, как учитывать страхование и техническое обслуживание основных средств.

Разделы этой статьи содержат ссылки на дополнительную информацию о том, как настроить основные средства. После завершения настройки можно приступать к работе с основными средствами. Дополнительные сведения см. в разделе [Использование основных средств](fa-manage.md).  

> [!NOTE]  
> Вы можете записывать транзакции по основным средствам на странице **Журнал ГК учета основных средств** или на странице **Журнал ОС** в зависимости от назначения транзакций: для финансовой отчетности или для внутреннего управления. Справочные статьи по основным средствам описывают только использование страницы **Журнал ГК учета основных средств**.  

Если вы включаете действие основного средства в разделе **Интеграция С ГК** на странице **Карточка книги амортизации**, страница **Журнал ГК учета основных средств** используется для учета транзакций для соответствующего действия.
