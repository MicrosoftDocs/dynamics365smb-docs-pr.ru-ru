### <a name="to-export-a-report-layout-to-a-file"></a><span data-ttu-id="fbb4f-108">Экспорт макета отчета в файл</span><span class="sxs-lookup"><span data-stu-id="fbb4f-108">To export a report layout to a file</span></span>  

1.  <span data-ttu-id="fbb4f-109">Выберите значок ![Лампочка, которая открывает функцию "Что вы хотите сделать"](media/ui-search/search_small.png "Что вы хотите сделать"), введите **Выбор макета отчета**, затем выберите соответствующую ссылку.</span><span class="sxs-lookup"><span data-stu-id="fbb4f-109">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  

2.  <span data-ttu-id="fbb4f-110">Выберите строку отчета, которая содержит пользовательский макет отчета, который требуется экспортировать, затем выберите действие **Пользовательские макеты**.</span><span class="sxs-lookup"><span data-stu-id="fbb4f-110">Select the row for the report that contains the custom report layout that you want to export, and then choose the **Custom Layouts** action.</span></span>  

3.  <span data-ttu-id="fbb4f-111">На странице **Макеты отчета** выберите макет отчета, который требуется экспортировать в файл, затем выберите действие **Экспортировать макет**.</span><span class="sxs-lookup"><span data-stu-id="fbb4f-111">On the **Report Layouts** page, select the report layout that you want to export to a file, and then choose the **Export Layout** action.</span></span>  

4.  <span data-ttu-id="fbb4f-112">В диалоговом окне **Экспорт файла** выберите кнопку **Сохранить** и сохраните файл в расположение на компьютере или в сети.</span><span class="sxs-lookup"><span data-stu-id="fbb4f-112">In the **Export File** dialog box, choose the **Save** button, and then save the file to a location on your computer or network.</span></span>  

### <a name="to-import-a-report-layout-file"></a><span data-ttu-id="fbb4f-113">Импорт файла макета отчета</span><span class="sxs-lookup"><span data-stu-id="fbb4f-113">To import a report layout file</span></span>  

1.  <span data-ttu-id="fbb4f-114">Убедитесь, что соответствующий файл, определяющий макет отчета, доступен на компьютере или в сети.</span><span class="sxs-lookup"><span data-stu-id="fbb4f-114">Make sure that the relevant file that defines the report layout is available on your computer or network.</span></span>  

     <span data-ttu-id="fbb4f-115">Для макетов отчета Word файл должен иметь расширение DOCX.</span><span class="sxs-lookup"><span data-stu-id="fbb4f-115">A Word report layout file must have the .docx file type extension.</span></span> <span data-ttu-id="fbb4f-116">Для макетов отчета RDLC файл должен иметь расширение RDLC или RDL.</span><span class="sxs-lookup"><span data-stu-id="fbb4f-116">An RDLC report layout file must have the .rdlc or .rdl file type extension.</span></span>  

2.  <span data-ttu-id="fbb4f-117">Выберите значок ![Лампочка, которая открывает функцию "Что вы хотите сделать"](media/ui-search/search_small.png "Что вы хотите сделать"), введите **Выбор макета отчета**, затем выберите соответствующую ссылку.</span><span class="sxs-lookup"><span data-stu-id="fbb4f-117">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  

3.  <span data-ttu-id="fbb4f-118">Выберите строку отчета, для которого требуется импортировать макет отчета, затем выберите действие **Пользовательские макеты**.</span><span class="sxs-lookup"><span data-stu-id="fbb4f-118">Select the row for the report to which you want to import the report layout, and then choose the **Custom Layouts** action.</span></span>  

4.  <span data-ttu-id="fbb4f-119">На странице **Макеты отчета** выберите макет отчета, в который требуется импортировать файл, затем выберите действие **Импортировать макет**.</span><span class="sxs-lookup"><span data-stu-id="fbb4f-119">On the **Report Layouts** page, select the report layout to which you want to import the file, and then choose the **Import Layout** action.</span></span>  

5.  <span data-ttu-id="fbb4f-120">В диалоговом окне **Импорт** выберите документ, определяющий макет отчета, и выберите кнопку **Открыть**.</span><span class="sxs-lookup"><span data-stu-id="fbb4f-120">In the **Import** dialog box, select the document that defines the report layout, and then choose the **Open** button.</span></span>  

 <span data-ttu-id="fbb4f-121">Исходный пользовательский макет отчета заменяется на импортированный макет отчета.</span><span class="sxs-lookup"><span data-stu-id="fbb4f-121">The original custom report layout is replaced with the imported report layout.</span></span>
