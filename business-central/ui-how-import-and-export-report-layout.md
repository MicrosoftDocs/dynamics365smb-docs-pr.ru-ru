<span data-ttu-id="80785-104">Можно импортировать и экспортировать существующий пользовательский макет отчета в виде файла в расположении на компьютере или в сети.</span><span class="sxs-lookup"><span data-stu-id="80785-104">You can import and export an existing custom report layout as a file to and from a location on your computer and network.</span></span> <span data-ttu-id="80785-105">Например, можно экспортировать макет отчета, а затем отправить файл другому лицу для изменения.</span><span class="sxs-lookup"><span data-stu-id="80785-105">For example, you can export a report layout, and then send the file to another person to modify.</span></span> <span data-ttu-id="80785-106">Это лицо может затем внести изменения в макет и вернуть файл, чтобы вы могли импортировать его обратно.</span><span class="sxs-lookup"><span data-stu-id="80785-106">That person can then make the modifications to layout and return the file to you so that you can import it back.</span></span>  

> [!IMPORTANT]  
>  <span data-ttu-id="80785-107">Невозможно импортировать экспортировать встроенные макеты отчетов.</span><span class="sxs-lookup"><span data-stu-id="80785-107">You cannot import or export built-in report layouts.</span></span>  

### <a name="to-export-a-report-layout-to-a-file"></a><span data-ttu-id="80785-108">Экспорт макета отчета в файл</span><span class="sxs-lookup"><span data-stu-id="80785-108">To export a report layout to a file</span></span>  

1.  <span data-ttu-id="80785-109">Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать](media/ui-search/search_small.png "Что вы хотите сделать"), введите **Выбор макета отчета**, затем выберите связанную ссылку.</span><span class="sxs-lookup"><span data-stu-id="80785-109">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  

2.  <span data-ttu-id="80785-110">Выберите строку отчета, содержащего пользовательский макет отчета, который требуется экспортировать, а затем на вкладке **Главная** в группе **Процесс** выберите **Пользовательские макеты**.</span><span class="sxs-lookup"><span data-stu-id="80785-110">Select the row for the report that contains the custom report layout that you want to export, and then on the **Home** tab, in the **Process** group, choose **Custom Layouts**.</span></span>  

3.  <span data-ttu-id="80785-111">На странице **Макеты отчета** выберите макет отчета, который требуется экспортировать в файл, а затем на вкладке **Главная** в группе **Процесс** выберите **Экспортировать макет**.</span><span class="sxs-lookup"><span data-stu-id="80785-111">On the **Report Layouts** page, select the report layout that you want to export to a file, and then on the **Home** tab, in the **Process** group, choose **Export Layout**.</span></span>  

4.  <span data-ttu-id="80785-112">В диалоговом окне **Экспорт файла** щелкните **Сохранить** и сохраните файл в расположение на компьютере или в сети.</span><span class="sxs-lookup"><span data-stu-id="80785-112">In the **Export File** dialog box, choose **Save**, and then save the file to a location on your computer or network.</span></span>  

### <a name="to-import-a-report-layout-file"></a><span data-ttu-id="80785-113">Импорт файла макета отчета</span><span class="sxs-lookup"><span data-stu-id="80785-113">To import a report layout file</span></span>  

1.  <span data-ttu-id="80785-114">Убедитесь, что соответствующий файл, определяющий макет отчета, доступен на компьютере или в сети.</span><span class="sxs-lookup"><span data-stu-id="80785-114">Make sure that the relevant file that defines the report layout is available on your computer or network.</span></span>  

     <span data-ttu-id="80785-115">Для макетов отчета Word файл должен иметь расширение DOCX.</span><span class="sxs-lookup"><span data-stu-id="80785-115">A Word report layout file must have the .docx file type extension.</span></span> <span data-ttu-id="80785-116">Для макетов отчета RDLC файл должен иметь расширение RDLC или RDL.</span><span class="sxs-lookup"><span data-stu-id="80785-116">An RDLC report layout file must have the .rdlc or .rdl file type extension.</span></span>  

2.  <span data-ttu-id="80785-117">Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать](media/ui-search/search_small.png "Что вы хотите сделать"), введите **Выбор макета отчета**, затем выберите связанную ссылку.</span><span class="sxs-lookup"><span data-stu-id="80785-117">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  

3.  <span data-ttu-id="80785-118">Выберите строку отчета, для которого требуется импортировать макет отчета, а затем на вкладке **Главная** в группе **Процесс** выберите **Пользовательские макеты**.</span><span class="sxs-lookup"><span data-stu-id="80785-118">Select the row for the report to which you want to import the report layout, and then on the **Home** tab, in the **Process** group, choose **Custom Layouts**.</span></span>  

4.  <span data-ttu-id="80785-119">На странице **Макеты отчета** выберите макет отчета, в который требуется экспортировать файл, а затем на вкладке **Главная** в группе **Процесс** выберите **Импортировать макет**.</span><span class="sxs-lookup"><span data-stu-id="80785-119">On the **Report Layouts** page, select the report layout to which you want to import the file, and then on the **Home** tab, in the **Process** group, choose **Import Layout**.</span></span>  

5.  <span data-ttu-id="80785-120">В диалоговом окне **Импорт** выберите документ, определяющий макет отчета, и нажмите кнопку **Открыть**.</span><span class="sxs-lookup"><span data-stu-id="80785-120">In the **Import** dialog box, select the document that defines the report layout, and then choose **Open**.</span></span>  

 <span data-ttu-id="80785-121">Исходный пользовательский макет отчета заменяется на импортированный макет отчета.</span><span class="sxs-lookup"><span data-stu-id="80785-121">The original custom report layout is replaced with the imported report layout.</span></span>
