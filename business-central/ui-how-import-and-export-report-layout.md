# <a name="import-and-export-a-report-or-document-layout"></a><span data-ttu-id="2a0c0-103">Импорт и экспорт макета отчета или документа</span><span class="sxs-lookup"><span data-stu-id="2a0c0-103">Import and Export a Report or Document Layout</span></span>
<span data-ttu-id="2a0c0-104">Можно импортировать и экспортировать существующий пользовательский макет отчета в виде файла в расположении на компьютере или в сети.</span><span class="sxs-lookup"><span data-stu-id="2a0c0-104">You can import and export an existing custom report layout as a file to and from a location on your computer and network.</span></span> <span data-ttu-id="2a0c0-105">Например, можно экспортировать макет отчета, а затем отправить файл другому лицу для изменения.</span><span class="sxs-lookup"><span data-stu-id="2a0c0-105">For example, you can export a report layout, and then send the file to another person to modify.</span></span> <span data-ttu-id="2a0c0-106">Это лицо может затем внести изменения в макет и вернуть файл, чтобы вы могли импортировать его обратно.</span><span class="sxs-lookup"><span data-stu-id="2a0c0-106">That person can then make the modifications to layout and return the file to you so that you can import it back.</span></span>  

> [!IMPORTANT]  
>  <span data-ttu-id="2a0c0-107">Невозможно импортировать экспортировать встроенные макеты отчетов.</span><span class="sxs-lookup"><span data-stu-id="2a0c0-107">You cannot import or export built-in report layouts.</span></span>  

### <a name="to-export-a-report-layout-to-a-file"></a><span data-ttu-id="2a0c0-108">Экспорт макета отчета в файл</span><span class="sxs-lookup"><span data-stu-id="2a0c0-108">To export a report layout to a file</span></span>  

1.  <span data-ttu-id="2a0c0-109">Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать](media/ui-search/search_small.png "Что вы хотите сделать"), введите **Выбор макета отчета**, а затем выберите соответствующую ссылку.</span><span class="sxs-lookup"><span data-stu-id="2a0c0-109">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  

2.  <span data-ttu-id="2a0c0-110">Выберите строку отчета, которая содержит пользовательский макет отчета, который требуется экспортировать, затем выберите действие **Пользовательские макеты**.</span><span class="sxs-lookup"><span data-stu-id="2a0c0-110">Select the row for the report that contains the custom report layout that you want to export, and then choose the **Custom Layouts** action.</span></span>  

3.  <span data-ttu-id="2a0c0-111">На странице **Макеты отчета** выберите макет отчета, который требуется экспортировать в файл, затем выберите действие **Экспортировать макет**.</span><span class="sxs-lookup"><span data-stu-id="2a0c0-111">On the **Report Layouts** page, select the report layout that you want to export to a file, and then choose the **Export Layout** action.</span></span>  

4.  <span data-ttu-id="2a0c0-112">В диалоговом окне **Экспорт файла** выберите кнопку **Сохранить** и сохраните файл в расположение на компьютере или в сети.</span><span class="sxs-lookup"><span data-stu-id="2a0c0-112">In the **Export File** dialog box, choose the **Save** button, and then save the file to a location on your computer or network.</span></span>  

### <a name="to-import-a-report-layout-file"></a><span data-ttu-id="2a0c0-113">Импорт файла макета отчета</span><span class="sxs-lookup"><span data-stu-id="2a0c0-113">To import a report layout file</span></span>  

1.  <span data-ttu-id="2a0c0-114">Убедитесь, что соответствующий файл, определяющий макет отчета, доступен на компьютере или в сети.</span><span class="sxs-lookup"><span data-stu-id="2a0c0-114">Make sure that the relevant file that defines the report layout is available on your computer or network.</span></span>  

     <span data-ttu-id="2a0c0-115">Для макетов отчета Word файл должен иметь расширение DOCX.</span><span class="sxs-lookup"><span data-stu-id="2a0c0-115">A Word report layout file must have the .docx file type extension.</span></span> <span data-ttu-id="2a0c0-116">Для макетов отчета RDLC файл должен иметь расширение RDLC или RDL.</span><span class="sxs-lookup"><span data-stu-id="2a0c0-116">An RDLC report layout file must have the .rdlc or .rdl file type extension.</span></span>  

2.  <span data-ttu-id="2a0c0-117">Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать](media/ui-search/search_small.png "Что вы хотите сделать"), введите **Выбор макета отчета**, а затем выберите соответствующую ссылку.</span><span class="sxs-lookup"><span data-stu-id="2a0c0-117">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  

3.  <span data-ttu-id="2a0c0-118">Выберите строку отчета, для которого требуется импортировать макет отчета, затем выберите действие **Пользовательские макеты**.</span><span class="sxs-lookup"><span data-stu-id="2a0c0-118">Select the row for the report to which you want to import the report layout, and then choose the **Custom Layouts** action.</span></span>  

4.  <span data-ttu-id="2a0c0-119">На странице **Макеты отчета** выберите макет отчета, в который требуется импортировать файл, затем выберите действие **Импортировать макет**.</span><span class="sxs-lookup"><span data-stu-id="2a0c0-119">On the **Report Layouts** page, select the report layout to which you want to import the file, and then choose the **Import Layout** action.</span></span>  

5.  <span data-ttu-id="2a0c0-120">В диалоговом окне **Импорт** выберите документ, определяющий макет отчета, и выберите кнопку **Открыть**.</span><span class="sxs-lookup"><span data-stu-id="2a0c0-120">In the **Import** dialog box, select the document that defines the report layout, and then choose the **Open** button.</span></span>  

 <span data-ttu-id="2a0c0-121">Исходный пользовательский макет отчета заменяется на импортированный макет отчета.</span><span class="sxs-lookup"><span data-stu-id="2a0c0-121">The original custom report layout is replaced with the imported report layout.</span></span>
