# <a name="move-items-in-advanced-warehouse-configurations-that-use-directed-put-away-and-pick"></a>Перемещение товаров в расширенных конфигурациях склада с использованием расширенного размещение и подбора

Вы можете перемещать товары между ячейками без спроса из исходного документа. Например, вы можете сделать это в рамках следующих действий:

* Реорганизуйте ваш склад.
* Доставьте товары в зону досмотра.
* Возьмите товары из ячеек подбора временно, например, чтобы послужить демонстрационной моделью на презентации продаж.
* Переместите дополнительные товары в производственную зону для компонентов, сконфигурированных с некоторыми методами списания.
* Переместите произведенные или собранные товары из зоны производства или сборки на склад.
* Переместите товары из зоны массового хранения в ячейки, которые вы используете для подбора.

Способ перемещения товаров зависит от того, как настроен склад. [Настройка управления складом](warehouse-setup-warehouse.md).

В конфигурациях склада, в которых для складов включен переключатель **Направленные подбор и размещение**, незапланированные перемещения можно зарегистрировать на следующих страницах:  

* Журнал перемещения
* Внутренний складской подбор
* Внутреннее складское размещение
* Журнал реклассификации склада

Страницы **Журнал перемещения**, **Внутренний складской подбор** и **Внутреннее складское размещение** работают таким же образом. Используйте страницы для подготовки складских операций для сотрудников склада. Разница заключается в расширенных функциях, связанных с каждой страницей, и в различных типах складских операций, которые, вероятно, выполняются разными сотрудниками:

* Журнал перемещения позволяет заполнять ячейки подбора с высоким рейтингом товарами из других ячеек
* Размещение использует шаблоны размещения
* Подбор использует ранжирование и доступность ячеек

## <a name="warehouse-movement-worksheet"></a>Журнал перемещения склада

### <a name="to-move-items-with-the-warehouse-movement-worksheet"></a>Перемещение товаров с помощью журнала складских передвижений

1. Выберите значок ![Лампочка, которая открывает функцию «Что вы хотите сделать»](media/ui-search/search_small.png "Что вы хотите сделать"), значок, введите **Журнал перемещения**, а затем выберите связанную ссылку.  
2. Заполните поля в строках журнала вручную или используйте одно из следующих действий для автоматического заполнения строк:

    * **Получить содержимое ячейки** вводит в строки журнала все содержимое указанной ячейки или ячеек.
    * **Расчет пополнения ячеек** используя рейтинги ячеек, предлагает пополнить ячейки с высоким рейтингом за счет ячеек с низким рейтингом.

    > [!NOTE]  
    > Если товар соответствует критериям в списке ниже, поля **Из зоны** и **Из корзины** будут пустыми. [!INCLUDE [prod_short](includes/prod_short.md)] вычисляет, откуда перемещать товары, только когда вы используете действие **Создать перемещение**.  
    >  
    > * Указан срок годности товара.  
    > * Включен переключатель **Выбрать по методу FEFO** в карточке склада.  
    > * Вы используете функцию **Расчет пополнения ячеек**.  

3. Выберите действие **Создать перемещение**, чтобы создать перемещение. Когда перемещение будет готов, его можно зарегистрировать.  

### <a name="to-register-the-warehouse-movement"></a>Регистрация складского передвижения

1. Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **Перемещения**, а затем выберите связанную ссылку.  
2. Откройте документ перемещения для регистрации.  
3. В строках **Поместить** укажите место, время и единицу товара, который нужно переместить путем изменения полей **Код зоны**, **Код ячейки**, **Кол-во для обработки** или **Дата выполнения**.  
4. В строках **Взять** в поле **Кол-во для обработки** укажите количество содержимого ячейки, которую необходимо переместить. Вы можете редактировать это поле только в строках **Взять**. 

    > [!NOTE]
    > Если товары для одной строки необходимо подобрать или поместить в несколько ячеек, например из-за того, что назначенная ячейка полная, то воспользуйтесь функцией **Разделить строку** на экспресс-вкладке **Строки**. Действие создает строку для обработки оставшегося количества.
 
5. Чтобы зарегистрировать все предложенные количества как указано в поле **Кол-во**, выберите действие **Автозаполнение кол-ва для обработки**.  
6. Выберите действие **Регистр**.  

> [!NOTE]  
> Для складов, в которых используется направленное размещение и подбор, вы не можете вручную перемещать товары в ячейки типа **ПРИЕМ**, так как они еще не считаются доступными запасами. Вы должны убрать товары из этих ячеек, прежде чем они станут доступны для перемещения.

## <a name="internal-pick"></a>Внутренний подбор

### <a name="to-create-an-internal-pick"></a>Создание внутренней подборки

1. Выберите значок ![Лампочка, которая открывает функцию «Что вы хотите сделать»](media/ui-search/search_small.png "Что вы хотите сделать"), значок, введите **Внутренний подбор склада**, а затем выберите связанную ссылку.  
2. Выберите действие **Создать**.
3. Заполните поле **Номер** поле **Код склада** и поле **В код ячейки** на экспресс-вкладке **Общее**. Поле **В код ячейки** определяет ячейку, в которую передаются подобранные товары. Для производственных целей эта ячейка будет ячейкой входящего производства или ячейкой сборочного цеха. Для других целей следует выбрать код ячейки, который не используется для подбора, чаще всего это ячейка этапа, отгрузки или используемая для специальных целей.  
4. Выбрать товар в поле **Код товара** и ввести подлежащее подбору количество.  
5. Выберите действие **Создать подбор**. Складская инструкция по подбору готова для выполнения ее работником склада. В качестве альтернативы выбрать действие **Выпуск** и создать складские подборы с помощью страницы **Журнал подбора**. Чтобы узнать больше о журналах подбора, перейдите к [Массовое создание документов подбора с журналом подбора](warehouse-how-to-pick-items-for-warehouse-shipment.md#to-create-pick-documents-in-bulk-with-the-pick-worksheet).
6. Когда подбор будет готов, его можно зарегистрировать.  

### <a name="to-register-the-warehouse-pick"></a>Регистрация складского подбора

1. Выберите значок ![Лампочка, которая открывает функцию «Что вы хотите сделать»](media/ui-search/search_small.png "Что вы хотите сделать"), значок, введите **Складской подбор**, а затем выберите связанную ссылку.  

    Чтобы работать с определенным подбором, выберите этот подбор в списке или отфильтруйте список, чтобы найти подборы, назначенные вам.  
2. Если поле **Код исполнителя** пустое, при необходимости введите для идентификации ваш ИД.  
3. Подбор товаров.  

    Поскольку склад настроен на использование расширенного подбора и размещения, ранжирование ячеек определяет ячейки для подбора. Эти ячейки предлагаются в строках подбора. Инструкции содержат как минимум две раздельных строк для действий "Взять" и "Поместить".  

4. После выполнения подбора и помещения товаров в зону отгрузки или ячейку отгрузки выберите действие **Зарегистрировать подбор**.  

## <a name="internal-put-away"></a>Внутреннее размещение

### <a name="to-create-an-internal-put-away"></a>Создание внутреннего размещения

1. Выберите значок ![Лампочка, которая открывает функцию «Что вы хотите сделать»](media/ui-search/search_small.png "Что вы хотите сделать"), значок, введите **Внутренние складские размещения**, а затем выберите связанную ссылку.  
2. Выберите действие **Создать**.
3. Заполните поле **Номер** и **Код склада**.
4. Заполните строку для каждого подлежащего передвижению на склад товара. Поля **Номер товара** и **Количество** обязательны.

    > [!NOTE]  
    > При выборе поля **Код товара** откроется страница **Содержимое ячейки**, а не страница **Товары**. Это вызвано тем, что необходимо разместить товар, находящийся в определенной ячейке — *Содержимое ячейки*, а не просто товар, и вам уже известно, из какой ячейки необходимо взять товар. Если вы заполнили поле **Из кода ячейки**, содержимое ячейки будет отфильтровано по этому значению.

5. Чтобы в строки ввести содержимое всей ячейки или отфильтрованное содержимое ячеек на складе, выберите действие **Получить содержимое ячейки**.  
6. Выберите действие **Создать размещение**. Складская инструкция по размещению готова для сотрудника склада. В качестве альтернативы выбрать действие **Выпуск** и создать складские размещения с помощью страницы **Журнал размещения**. Чтобы узнать больше о журналах размещения, перейдите к [Массовое создание документов размещения с журналом размещения](warehouse-how-to-put-items-away-with-warehouse-put-aways.md#to-create-put-away-documents-in-bulk-with-the-put-away-worksheet).
6. Когда размещение будет готово, его можно зарегистрировать.  

### <a name="to-register-the-warehouse-put-away"></a>Чтобы зарегистрировать складское размещение

1. Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **Размещения**, а затем выберите связанную ссылку.
2. Откройте складское размещение, которое необходимо обработать.  
3. При необходимости введите свой ИД пользователя при начале работы над размещением.  

    Для оптимизации процесса размещения можно сортировать строки размещения по различным критериям. Например, по товару, номеру полки или сроку выполнения.

    * Если строки "Взять" и "Поместить" для каждой приемки не следуют непосредственно одна за другой, а требуется, чтобы они располагались в таком порядке, отсортируйте строки, выбрав значение **Товар** в поле **Метод сортировки**.  
    * Если ранжирование ячеек отражает физический макет склада, используйте метод сортировки **Рейтинг ячеек**, чтобы организовать обходной путь склада ячеек.  

4. Выполните размещение.

    Каждая строка внутреннего размещения в складском размещении делится не менее, чем на две.  

    * Первая строка со значением **Взять** в поле **Тип действия** указывает, где товары расположены в зоне приемки. В этой строке нельзя изменить зону и ячейку.  
    * Следующая строка с типом действия **Поместить** в поле **Тип действия** указывает, где поместить товары на складе. Если вы получили большое количество товара по одной строке приходной накладной, то они могут быть размещены в несколько ячеек, поскольку в каждой ячейке есть строка Поместить.  

5. После того как все товары размещены в ячейках согласно инструкции, выберите действие **Зарегистрировать размещение**.  

## <a name="to-register-a-movement-that-has-already-happened"></a>Чтобы зарегистрировать перемещение, которое уже произошло

Если вам необходимо зарегистрировать тот факт, что товары уже были перемещены в другие ячейки без размещения, подбора или перемещения, вы можете использовать страницу **Журнал реклассификаций склада** для регистрации перемещения.

1. Выберите значок ![Лампочка, которая открывает функцию «Что вы хотите сделать»](media/ui-search/search_small.png "Что вы хотите сделать"), значок, введите **Журнал реклассификаций склада**, а затем выберите связанную ссылку.  
2. Заполните поля **Номер товара**, **Из кода зоны**, **Из кода ячейки**, **В код зоны** и **В код ячейки**.  
3. Выберите действие **Регистр**.
