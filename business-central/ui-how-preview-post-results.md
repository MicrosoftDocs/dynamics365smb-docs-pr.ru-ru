# <a name="preview-posting-results"></a><span data-ttu-id="3fd29-103">Предварительный просмотр результатов учета</span><span class="sxs-lookup"><span data-stu-id="3fd29-103">Preview Posting Results</span></span>
<span data-ttu-id="3fd29-104">Во всех документах и журналах, которые можно учитывать, можно нажать кнопку **Предварительный просмотр учета** для просмотра различных типов операций, которые будут созданы при учете документа или журнала.</span><span class="sxs-lookup"><span data-stu-id="3fd29-104">On every document and journal that can be posted, you can choose the **Preview Posting** button to review the different types of entries that will be created when you post the document or journal.</span></span>

## <a name="to-preview-gl-entries-that-will-result-from-posting-a-purchase-invoice"></a><span data-ttu-id="3fd29-105">Просмотр операций ГК, которые появятся в результате учета счета покупки</span><span class="sxs-lookup"><span data-stu-id="3fd29-105">To preview G/L entries that will result from posting a purchase invoice</span></span>
1. <span data-ttu-id="3fd29-106">Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать](media/ui-search/search_small.png "Что вы хотите сделать"), введите **Счета покупки**, затем выберите связанную ссылку.</span><span class="sxs-lookup"><span data-stu-id="3fd29-106">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Purchase Invoices**, and then choose the related link.</span></span>
2. <span data-ttu-id="3fd29-107">Создайте счет покупки.</span><span class="sxs-lookup"><span data-stu-id="3fd29-107">Create a purchase invoice.</span></span> <span data-ttu-id="3fd29-108">Дополнительные сведения см. в разделе [Регистрация покупок](purchasing-how-record-purchases.md).</span><span class="sxs-lookup"><span data-stu-id="3fd29-108">For more information, see [Record Purchases](purchasing-how-record-purchases.md).</span></span>
3. <span data-ttu-id="3fd29-109">Выберите **Предварительный просмотр учета**.</span><span class="sxs-lookup"><span data-stu-id="3fd29-109">Choose **Preview Posting**.</span></span>
4. <span data-ttu-id="3fd29-110">На странице **Предварительный просмотр учета** выберите **Операция ГК**, а затем выберите **Показать связанные операции**.</span><span class="sxs-lookup"><span data-stu-id="3fd29-110">On the **Posting Preview** page, select **G/L Entry**, and then choose **Show Related Entries**.</span></span>  
   <span data-ttu-id="3fd29-111">На странице **Предварительный просмотр операций ГК** будут показаны операции, которые будут созданы в результате учета счета покупки.</span><span class="sxs-lookup"><span data-stu-id="3fd29-111">The **G/L Entries Preview** page shows which entries will be created when you post the purchase invoice.</span></span>
