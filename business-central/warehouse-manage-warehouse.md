<span data-ttu-id="c41f5-108">На крупных складах эти различные задачи обработки могут распределяться по отделам, а управление их интеграцией осуществляется с помощью расширенного рабочего процесса.</span><span class="sxs-lookup"><span data-stu-id="c41f5-108">In large warehouses, these different handling tasks can be separated by departments and the integration managed by a directed workflow.</span></span> <span data-ttu-id="c41f5-109">В более простых структурах процесс менее формализован, а складские операции выполняются с использованием т. н. операций размещения запасов и подбора запасов.</span><span class="sxs-lookup"><span data-stu-id="c41f5-109">In simpler installations, the flow is less formalized and the warehouse activities are performed with so-called inventory put-aways and inventory picks.</span></span> <span data-ttu-id="c41f5-110">Дополнительные сведения о сравнении базовых и расширенных конфигурациях складов см. в разделах [Сведения о проектировании: обзор склада](design-details-warehouse-overview.md).</span><span class="sxs-lookup"><span data-stu-id="c41f5-110">For more information about basic versus advanced warehouse configurations, see [Design Details: Warehouse Overview](design-details-warehouse-overview.md).</span></span>

<span data-ttu-id="c41f5-111">Прежде чем можно будет осуществлять складские операции, необходимо настроить систему для соответствующей сложности складской обработки.</span><span class="sxs-lookup"><span data-stu-id="c41f5-111">Before you can perform warehouse activities, you must set the system up for the relevant complexity of warehouse processing.</span></span> <span data-ttu-id="c41f5-112">Дополнительные сведения см. в разделе [Настройка управления складом](warehouse-setup-warehouse.md).</span><span class="sxs-lookup"><span data-stu-id="c41f5-112">For more information, see [Setting Up Warehouse Management](warehouse-setup-warehouse.md).</span></span>
