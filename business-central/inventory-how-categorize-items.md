## <a name="to-create-an-item-category"></a><span data-ttu-id="308e0-107">Создание категории товара</span><span class="sxs-lookup"><span data-stu-id="308e0-107">To create an item category</span></span>
1. <span data-ttu-id="308e0-108">Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать](media/ui-search/search_small.png "Что вы хотите сделать"), введите **Категории товара**, затем выберите связанную ссылку.</span><span class="sxs-lookup"><span data-stu-id="308e0-108">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Item Categories**, and then choose the related link.</span></span>
2. <span data-ttu-id="308e0-109">В окне **Категории товара** выберите действие **Создать**.</span><span class="sxs-lookup"><span data-stu-id="308e0-109">In the **Item Categories** window, choose the **New** action.</span></span>
3. <span data-ttu-id="308e0-110">В окне **Карточка категории товаров** заполните требуемые поля на экспресс-вкладке **Общее**.</span><span class="sxs-lookup"><span data-stu-id="308e0-110">In the **Item Category Card** window, on the **General** FastTab, fill in the fields as necessary.</span></span> [!INCLUDE[tooltip-inline-tip](includes/tooltip-inline-tip_md.md)]
4. <span data-ttu-id="308e0-111">На экспресс-вкладке **Атрибуты** укажите все атрибуты товаров для категории товара.</span><span class="sxs-lookup"><span data-stu-id="308e0-111">On the **Attributes** FastTab, specify any item attributes for the item category.</span></span> <span data-ttu-id="308e0-112">Дополнительные сведения см. в пункте "Назначение атрибутов товара категории товаров" раздела [Работа с атрибутами продуктов](inventory-how-work-item-attributes.md).</span><span class="sxs-lookup"><span data-stu-id="308e0-112">For more information, see the "To assign item attributes to an item category" section in [Work with Item Attributes](inventory-how-work-item-attributes.md).</span></span>

> [!NOTE]  
>   <span data-ttu-id="308e0-113">Если категория товаров имеет родительскую категорию товаров, как указано в поле **Родительская категория**, то все атрибуты товаров, назначенные этой родительской категории, будут уже заполнены на экспресс-вкладке **Атрибуты**.</span><span class="sxs-lookup"><span data-stu-id="308e0-113">If the item category has a parent item category, as indicated by the **Parent Category** field, then any item attributes that are assigned to that parent item category are prefilled on the **Attributes** FastTab.</span></span>

> [!NOTE]  
>   <span data-ttu-id="308e0-114">Атрибуты товаров, назначенные категории товаров, автоматически применяются к товарам, которые назначены этой категории товаров.</span><span class="sxs-lookup"><span data-stu-id="308e0-114">Item attributes that you assign to an item category will automatically apply to the item that the item category is assigned to.</span></span>

## <a name="to-assign-an-item-category-to-an-item"></a><span data-ttu-id="308e0-115">Назначение товару категории товаров</span><span class="sxs-lookup"><span data-stu-id="308e0-115">To assign an item category to an item</span></span>
1. <span data-ttu-id="308e0-116">Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать](media/ui-search/search_small.png "Что вы хотите сделать"), введите **Товары**, затем выберите связанную ссылку.</span><span class="sxs-lookup"><span data-stu-id="308e0-116">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Items**, and then choose the related link.</span></span>
2. <span data-ttu-id="308e0-117">Откройте карточку товара, которому нужно присвоить категорию товаров.</span><span class="sxs-lookup"><span data-stu-id="308e0-117">Open the card for the item that you want to assign to an item category.</span></span>
3. <span data-ttu-id="308e0-118">Нажмите кнопку выбора в поле **Код товарной категории** и выберите существующую категорию товаров.</span><span class="sxs-lookup"><span data-stu-id="308e0-118">Choose the lookup button in the **Item Category Code** field and select an existing item category.</span></span> <span data-ttu-id="308e0-119">Можно также выбрать действие **Создать**, чтобы сначала создать новую категорию товаров, как объясняется в разделе "Создание категории товара".</span><span class="sxs-lookup"><span data-stu-id="308e0-119">Alternatively, choose the **New** action to first create a new item category as explained in the "To create an item category" section.</span></span>
