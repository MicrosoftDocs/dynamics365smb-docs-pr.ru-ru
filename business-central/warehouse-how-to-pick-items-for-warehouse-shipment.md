# <a name="pick-items-for-warehouse-shipment"></a>Подбор товаров для складской отгрузки

В [!INCLUDE[prod_short](includes/prod_short.md)] подберите и отгрузите товары с использованием одного из четырех методов, как описано в следующей таблице.

|Метод|Исходящий процесс|Требуется подбор|Требуется отгрузка|Уровень сложности (подробнее см. в разделе [Обзор Warehouse Management](design-details-warehouse-management.md))|  
|------|----------------|-----|---------|-------------------------------------------------------------------------------------|  
|А|Выполните учет подбора и отгрузки из строки заказа|||Нет отдельной складской операции.|  
|Б|Выполните учет подбора и отгрузки из складского документа подбора|Включено||База: последовательно по каждому заказу.|  
|Т|Выполните учет подбора и отгрузку из документа складской отгрузки||Включено|База: объединенная проводка получения/отправки для нескольких заказов.|  
|Д|Выполните учет подбора из складского документа подбора и учет отгрузки из документа складской отгрузки|Включено|Включено|Дополнительно|  

Подробнее см. в разделе [Исходящий складской поток](design-details-outbound-warehouse-flow.md).

Эта статья относится к методам Г в таблице. Чтобы узнать больше об отгрузке товаров, перейдите к [Сборка товаров](warehouse-how-ship-items.md).

Если склад настроен так, что требуется обработка как складского подбора, так и складской отгрузки, то для создания и обработки информации о подборе до учета складской отгрузки используются документы складского подбора.  

Вы не можете создать с нуля документ складского подбора. Подборы являются частью рабочего процесса, когда пользователь, обрабатывающий заказ, создает их в режиме распределения, или сотрудник склада создает их в режиме извлечения:

- В режиме извлечения, когда вы используете действие **Создать подбор** на странице **Складская отгрузка**. Выберите строки для подбора и подготовьте комплектование, указывая, например, из каких ячеек брать товар, в каких местах его размещать и сколько единиц товара обрабатывать. Ячейки могут быть предварительно определены для склада или ресурса.
- В режиме извлечения, когда вы используете действие **Выпуск** на странице **Складская отгрузка**, чтобы сделать товары доступными для сбор. Затем на странице **Журнал подбора** сотрудники склада могут использовать действие **Получить складские документы**, чтобы получить назначенный им подбор.

> [!NOTE]  
> Подбор для складской отгрузки товаров, которые собираются для заказа на продажу, выполняется так же, как для обычной отгрузки для складских подборов для отгрузки, как описано в этом статье. Количество строк подбора на количество для отгружаемого количества может быть много к одному, так как вы выбираете компоненты, а не сборочный элемент.  
>
> Строки складского подбора создаются для значения в поле **Остаток** в строках заказа на сборку, связанного со строкой заказа на продажу, который отгружается. Все компоненты подбираются одним действием. Подробнее см. в разделе [Обработка товаров сборки на заказ складских товаров в складских отгрузках](warehouse-how-ship-items.md#handling-assemble-to-order-items-in-warehouse-shipments).  
>  
> Для получения информации о подборе компонентов для заказов на сборку, в том числе о ситуациях когда сборочный элемент не должен быть в накладной продажи, см. раздел [Подбор для производства, сборки или работ в расширенных конфигурациях склада](warehouse-how-to-pick-for-internal-operations-in-advanced-warehousing.md).

## <a name="to-create-pick-documents-in-bulk-with-the-pick-worksheet"></a>Массовое создание документов подбора с журналом подбора

1. Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок введите **Журнал подбора**, а затем выберите связанную ссылку.  

2. Выберите действие **Получить складские документы**.  

    В список будут включены все отгрузки, переданные на подбор, в том числе те, для которых уже созданы инструкции по подбору. В этом списке не показаны документы со строками подбора, которые полностью подобраны и зарегистрированы.  
3. Выбрать отгрузку, для которой осуществляется подбор.

    > [!NOTE]  
    >  Если выбирается отгрузка или документ внутреннего подбора, инструкции для всех строк которых уже созданы, вы получите сообщение, которое говорит что-то вроде "нечего обрабатывать". Чтобы объединить инструкции по складскому подбору, которые вы уже создали, в одну инструкцию по подбору, вы должны сначала удалить отдельные складские подборы.

4. Для сортировки строк заполните поле **Метод сортировки**.  

    > [!NOTE]  
    >  Способ сортировки строк в журнале не распространяется автоматически на инструкцию по подбору. Однако существуют те же возможности для сортировки и ранжирования ячеек. Можно легко воссоздать порядок планирования строк в журнале при планировании инструкций по подбору или сортировкой в инструкциях по подбору.

5. Заполните поле **Кол-во для обработки** либо вручную, либо с помощью действия **Автозаполнение кол-ва для обработки**.  

    На странице показаны количества, доступные в ячейках переброски. Эта информация полезно для планирования рабочих заданий в ситуациях переброски. [!INCLUDE[prod_short](includes/prod_short.md)] всегда сначала будет предлагать подборку из ячейки переброски.
6. При необходимости можно отредактировать строки. Для того, чтобы сделать подбор более эффективным, можно также удалить строки. Например, если есть несколько строк с товарами, которые находятся в ячейках переброски, вы можете создать подборку для всех строк. Переброшенные товары будут отгружены с другими товарами по отгрузке, и в ячейках переброски появится место для новых поступивших товаров.  

    > [!NOTE]  
    >  Если вы удаляете строки, они удаляются только из этого журнала. Они не удаляются из списка выбора подбора.  

7. Выберите действие **Создать подбор**. Откроется страница **Создать подбор**, на которой можно добавить к подбору дополнительную информацию. Укажите, как объединить строки подбора в документах подбора путем выбора одного из следующих параметров.  

    |Параметр|Описание|
    |-|-|
    |По складу. Документ|Создает отдельные документы подбора для строк журнала с одинаковым складским исходным документом.|
    |По клиенту/поставщ./складу|Создайте отдельные документы подбора для каждого клиента (заказы на продажу), поставщика (заказы на возврат покупки) и склада (заказы на перемещение).|
    |По товару|Создайте отдельные документы подбора для каждого товара в журнале подбора.|
    |По исходной зоне|Создайте отдельные документы подбора для каждой зоны, из которой будут взяты товары.|
    |По ячейке|Создайте отдельные документы подбора для каждой ячейки, из которой будут взяты товары.|
    |По сроку выполнения|Создайте отдельные документы подбора для исходных документов с одинаковым сроком выполнения.|

    Следует определить способ создания документов подбора путем выбора одного из следующих параметров.

    |Параметр|Описание|
    |-|-|
    |Макс. Кол-во строк подбора|Создает документы подбора, которые охватывают не более указанного числа строк в каждом документе.|
    |Макс. Кол-во док. ист. подбора.|Создает документы подбора, каждый из которых охватывает не более указанного числа исходных документов.|
    |Код исполнителя|Создает документы подбора только для строк журнала, назначенных выбранному сотруднику склада.|
    |Метод сортировки для строк подбора|Выберите один из возможных параметров для сортировки строк в документе подбора.|
    |Установить фильтр разбивки|Скрывает промежуточные строки разбивки подбора, когда большая единица измерения преобразуется в более мелкую и завершается подбор.|
    |Не запол. кол-во для обработки|Поле "Кол-во для обработки" остается пустым в созданных строках подбора.|
    |Печать подбора|Печать документов подбора при их создании. Можно также печатать из созданных документов подбора.|

8. Выберите **ОК**. [!INCLUDE [prod_short](includes/prod_short.md)] создаст подбор в соответствии с вашим выбором.  

## <a name="to-pick-items-for-a-warehouse-shipment"></a>Складской подбор товаров для отгрузки

1. Выберите значок ![Лампочка, которая открывает функцию Что вы хотите сделать.](media/ui-search/search_small.png "Что вы хотите сделать") значок, введите **Складские подборы**, а затем выберите связанную ссылку.  

    Если необходимо работать с определенным подбором, выберите этот подбор в списке или отфильтруйте список, чтобы найти подборы, назначенные для вас. Откройте карточку подбора.  
2. Если поле **Код исполнителя** пустое, при необходимости введите для идентификации свой ИД.  
3. Подбор товаров.  

    Если склад настроен на использование ячеек, ячейки товаров по умолчанию используются для того, чтобы предложить, откуда взять товары. Инструкции содержат как минимум две раздельных строк для действий "Взять" и "Поместить".  

    Если склад настроен на использование расширенного подбора и размещения, ранжирование ячеек используется для вычисления наиболее подходящих для подбора ячеек. Эти ячейки предлагаются в строках подбора. Инструкции содержат как минимум две раздельных строк для действий "Взять" и "Поместить".  

    * Первая строка со значением **Взять** в поле **Тип действия** указывает, где товары расположены в зоне подбора. Если вы отгружаете большое количество товаров в одной строке отгрузки, вам, возможно, придется подбирать товары в нескольких ячейках, поэтому для каждой ячейки есть строка "Взять".
    * Следующая строка (с типом действия **Поместить** в поле **Тип действия**) указывает место размещения товаров на складе. В этой строке нельзя изменить зону и ячейку.

    > [!NOTE]
    > Если товары для одной строки необходимо подобрать или поместить в несколько ячеек, например из-за того, что назначенная ячейка полная, то воспользуйтесь функцией **Разделить строку** на экспресс-вкладке **Строки**. Действие создает строку для обработки оставшегося количества.

4. После выполнения подбора и помещения товаров в зону отгрузки или ячейку отгрузки выберите действие **Зарегистрировать подбор**.  

Теперь вы можете доставить товары в отгрузочный отсек и учесть отгрузку, включая соответствующий исходный документ, на странице **Складская отгрузка**. Подробнее см. в разделе [Отгрузка товаров](warehouse-how-ship-items.md).
