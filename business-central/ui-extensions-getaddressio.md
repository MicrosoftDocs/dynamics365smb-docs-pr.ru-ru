##<a name="to-use-the-getaddressio-uk-postcodes-extension-when-you-enter-an-address"></a><span data-ttu-id="bcaeb-107">Использование расширения "Почтовые индексы Великобритании GetAddress.io" при создании адреса</span><span class="sxs-lookup"><span data-stu-id="bcaeb-107">To use the GetAddress.io UK Postcodes extension when you enter an address</span></span>

    > [!NOTE]  
    >   <span data-ttu-id="bcaeb-109">Параметр **Найти адрес по почтовому индексу** доступен, только если поле **Код страны или региона** либо пусто, либо содержит значение **GB**.</span><span class="sxs-lookup"><span data-stu-id="bcaeb-109">The **Look up address from postcode** option is available only if the **Country/Region Code** field is either empty, or contains **GB**.</span></span>
