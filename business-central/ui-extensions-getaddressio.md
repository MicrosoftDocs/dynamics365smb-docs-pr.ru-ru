    > [!NOTE]  
    >   <span data-ttu-id="f25cb-109">Параметр **Найти адрес по почтовому индексу** доступен, только если поле **Код страны или региона** либо пусто, либо содержит значение **GB**.</span><span class="sxs-lookup"><span data-stu-id="f25cb-109">The **Look up address from postcode** option is available only if the **Country/Region Code** field is either empty, or contains **GB**.</span></span>
