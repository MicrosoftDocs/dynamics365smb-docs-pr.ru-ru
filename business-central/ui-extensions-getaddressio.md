<span data-ttu-id="46597-104">Расширение "Почтовые индексы Великобритании GetAddress.io" может помочь сэкономить время и избежать ошибок при вводе адресов для объектов, таких как клиенты, поставщики, банковские счета и сотрудники.</span><span class="sxs-lookup"><span data-stu-id="46597-104">The GetAddress.io UK Postcodes extension can save time and prevent mistakes when you enter addresses for entities like customers, vendors, bank accounts, and employees.</span></span> <span data-ttu-id="46597-105">Например, при создании клиента можно просто выбрать почтовый индекс, и [!INCLUDE[d365fin](includes/d365fin_md.md)] отобразит список адресов.</span><span class="sxs-lookup"><span data-stu-id="46597-105">For example, when you're creating a customer, you just choose a postcode and [!INCLUDE[d365fin](includes/d365fin_md.md)] shows a list of addresses.</span></span> <span data-ttu-id="46597-106">Выберите требуемый адрес, и поля адреса будут заполнены автоматически.</span><span class="sxs-lookup"><span data-stu-id="46597-106">Pick the address you want, and presto, the address fields are filled out.</span></span>  

##<a name="to-use-the-getaddressio-uk-postcodes-extension-when-you-enter-an-address"></a><span data-ttu-id="46597-107">Использование расширения "Почтовые индексы Великобритании GetAddress.io" при создании адреса</span><span class="sxs-lookup"><span data-stu-id="46597-107">To use the GetAddress.io UK Postcodes extension when you enter an address</span></span>
1. <span data-ttu-id="46597-108">В разделе **Адрес** выберите **Найти адрес по почтовому индексу**.</span><span class="sxs-lookup"><span data-stu-id="46597-108">Under **Address**, choose **Look up address from postcode**.</span></span>  

    > [!NOTE]  
    >   <span data-ttu-id="46597-109">Параметр **Найти адрес по почтовому индексу** доступен, только если поле **Код страны или региона** либо пусто, либо содержит значение **GB**.</span><span class="sxs-lookup"><span data-stu-id="46597-109">The **Look up address from postcode** option is available only if the **Country/Region Code** field is either empty, or contains **GB**.</span></span>
2. <span data-ttu-id="46597-110">В окне **Поиск почтового индекса** в поле **Почтовый индекс** введите или выберите почтовый индекс.</span><span class="sxs-lookup"><span data-stu-id="46597-110">In the **Postcode Search** window, in the **Postcode** field, enter or choose the postcode.</span></span>

> [!TIP]
