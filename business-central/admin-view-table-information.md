# <a name="viewing-table-information"></a>Просмотр информации о таблицах

Страница **8700 Сведения о таблице** содержит информацию обо всех системных и бизнес-таблицах в решении Business Central. В частности, на этой странице отображается информация о количестве данных, содержащихся в таблицах.

Эта информация может быть полезна для устранения проблем с производительностью, потому позволяет увидеть распределение объема данных по таблицам.

## <a name="viewing-table-information"></a>Просмотр информации о таблицах

Чтобы открыть эту страницу, выберите значок ![Поиск страницы или отчета](media/ui-search/search_small.png "Значок поиска страницы или отчета"), введите **Сведения о таблице**, а затем выберите соответствующую ссылку.

В следующей таблице показано, какая именно информация предоставляется по каждой таблице:

|Столбец|Описание|
|------|-----------|
|Название организации|Название организации, к которой относится таблица (если таблица относится к какой-либо организации).|
|Название таблицы|Название таблицы.|
|Номер таблицы|Идентификатор таблицы.|
|Кол-во записей|ОБщее количество записей, хранящихся в таблице.|
|Размер записи|Средний размер записи в килобайтах. Значение рассчитывается по следующей формуле: 1024(Размер)/(Кол-во записей). |
